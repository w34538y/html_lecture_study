10.5 RMI 의 어플리케이션 고유의 값

다음의 겉(표)에서는, RMI 로 이용되는 어플리케이션 고유의 값을 표현하는 비터미널 심볼을 나타냅니다. 이 표에서는, 각각의 심볼이 각각의 가지는 형태에 대응하고 있습니다. 각 심볼은 그것이 파묻히는 프로토콜을 사용해 정형됩니다.
