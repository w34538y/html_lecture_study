8.4 ServerRef 인터페이스

ServerRef 인터페이스는, 원격 객체의 구현에 대한 서버측의 핸들을 나타냅니다. 

package java.rmi.server;

public interface ServerRef extends RemoteRef {
	RemoteStub exportObject(java.rmi.Remote obj, Object data)
		throws java.rmi.RemoteException;
	String getClientHost() throws ServerNotActiveException;
}
exportObject 메소드는, 주어진 Remote 객체의 구현 obj 에 대한 클라이언트 Stub 객체를 찾든가 혹은 작성합니다. 파라미터 data 에는, 객체를 export 하는데 필요한 정보 (예를 들어 포트 번호)가 포함됩니다.

getClientHost 메소드는, 현재의 클라이언트의 호스트명을 돌려줍니다. 현재, 원격 메소드 호출을 조작중의 thread로부터 불려 갔을 경우에는, 호출을 실행중의 클라이언트의 호스트명이 돌려주어집니다. 원격 메소드에의 호출이 현재 실행중이 아니면,ServerNotActiveException 가 불려 갑니다.
