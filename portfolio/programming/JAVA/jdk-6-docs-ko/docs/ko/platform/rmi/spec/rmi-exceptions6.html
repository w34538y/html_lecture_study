A. 4 네이밍에 있어서의 예외

다음의 겉(표)는 java.rmi.Naming 클래스와 java.rmi.registry.Registry 인터페이스의 메소드로 발생하는 예외를 나타낸 것입니다.

예외
내용

java.rmi.AccessException

조작이 거부되었다. 레지스트리는 바인드, 리바인드 및 언바인드(unbind)를 동일 호스트상으로 한정한다. lookup 는 임의의 호스트로부터에서도 실행 가능

java.rmi.AlreadyBoundException

벌써 바인드 되고 있는 이름을 바인드 하려고 했다

java.rmi.NotBoundException

바인드되어 있지 않은 이름을 검색하려고 했다

java.rmi.UnknownHostException

미지의 호스트상의 레지스트리에 컨택트 하려고 했다
