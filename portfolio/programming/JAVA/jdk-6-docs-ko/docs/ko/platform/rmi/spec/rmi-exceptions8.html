내용

java.rmi.RMISecurityException

주 -이 예외는, Java 2 SDK, Standard Edition, v1. 2 에서는 추천 되고 있지 않습니다.

RMISecurityManager 에 의해 throw 된 시큐리티 예외

java.rmi.server.ServerCloneException

복제 (카피)에 실패했다

java.rmi.server.ServerNotActiveException

원격 서버-가 원격 메소드로 실행되어 있지 않을 때에 RemoteServer.getClientHost 메소드에 의해 클라이언트 호스트를 취득하려고 했다

java.rmi.server.SocketSecurityException

불법인 포트로부터 객체를 export 하려고 했다
