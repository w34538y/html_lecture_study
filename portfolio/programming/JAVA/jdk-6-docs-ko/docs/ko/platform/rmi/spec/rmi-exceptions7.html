A. 5 기동에 있어서의 예외

다음에, 기동 가능 객체에 관련하는 동작으로 throw 될 가능성이 있는, 예외의 일람을 나타냅니다. 기동 API 는 java.rmi.activation 패키지에 포함되어 있습니다.

예외
내용

java.rmi.activation.ActivateFailedException

기동 가능 객체에의 원격 호출로 기동에 실패했을 때에 RMI 런타임에 의해 throw 된다

java.rmi.activation.ActivationException

기동 인터페이스 및 클래스에 의해 사용되는 일반적인 예외 클래스

java.rmi.activation.UnknownGroupException

ActivationGroupID 파라미터 또는 ActivationGroupDesc 파라미터의 ActivationGroupID 가 무효가 되었을 때에, 기동 클래스 및 인터페이스의 메소드에 의해 throw 된다

java.rmi.activation.UnknownObjectException

ActivationID 파라미터가 무효가 되었을 때에, 기동 클래스 또는 인터페이스의 메소드에 의해 throw 된다
