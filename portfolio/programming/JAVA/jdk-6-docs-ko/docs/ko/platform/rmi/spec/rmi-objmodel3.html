2.2 용어의 정의

Java SE 플랫폼의 분산 객체 모델에 있어서의 「원격 객체」란, 다른 Java 가상 머신 (다른 호스트상에 있는 경우도 있다)으로부터 그 메소드를 호출할 수가 있는 객체입니다. 이 타입의 객체는, 원격 객체의 메소드를 선언하는 Java 인터페이스인, 1 개(살) 또는 복수의 「원격 인터페이스」에 의해 기술됩니다.

「원격 메소드 호출」 (RMI)이란, 원격 객체상에 있는 원격 인터페이스의 메소드를 호출하는 조작입니다. 이 조작에서 가장 중요한 것은, 원격 객체를 호출하기 위한 구문이, 로컬 객체의 메소드 호출의 경우와 같은 것입니다.
