레지스트리 인터페이스



제 6 장
RMI 시스템은,java.rmi.registry.Registry 인터페이스와 java.rmi.registry.LocateRegistry 클래스를 사용해, 객체를 단순명에 의해 꺼내거나 등록하거나 하는 일반적인 bootstrap 서비스를 실시합니다.

「레지스트리」는, 이름을 원격 객체에 맵하기 위한 원격 객체입니다. 어느 서버 프로세스도 자신의 레지스트리를 지원할 수가 있습니다. 혹은, 단독의 레지스트리를 호스트에게 사용할 수 있습니다.

LocateRegistry 메소드는, 특정의 호스트, 또는 호스트와 포트의 양쪽 모두로 동작하고 있는 레지스트리를 취득하기 위해서 사용합니다. java.rmi.Naming 클래스의 메소드는, 적절한 LocateRegistry.getRegistry 메소드를 사용해, Registry 인터페이스를 구현하는 원격 객체를 호출합니다.
