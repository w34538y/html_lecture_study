JavaTM Remote Method Invocation 의 스펙

JavaTM Remote Method Invocation (Java RMI)는, Java 언어용의 분산 객체 모델로, Java 플랫폼의 객체 모델의 시멘틱스를 보관 유지해, 분산 객체의 구현 및 사용을 용이하게 합니다. 이 시스템은, Modula-3 의 Network Objects 시스템과 Spring 의 하청계약의 특징적인 기능을 결합한 것으로, Java SE 플랫폼에 의해 실현되어 따분한가의 새로운 기능도 채용하고 있습니다.
