B0. 3 그 외의 프로퍼티

이러한 프로퍼티은, 구현 패키지에 포함되는, 특정의 구현 클래스가 놓여져 있는 장소를 지정하기 위해서 사용됩니다. 주:이러한 프로퍼티은, 모두, Java 2 SDK, Standard Edition, v1. 2 에서는 추천 되고 있지 않습니다.

프로퍼티
 설명

java.rmi.loader.packagePrefix

주 -이 예외는, Java 2 SDK, Standard Edition, v1. 2 에서는 추천 되고 있지 않습니다.

인터페이스 java.rmi.server.LoaderHandler 를 구현하는 클래스를 위한 패키지 접두사. 디폴트 값는 sun.rmi.server

java.rmi.registry.packagePrefix

주 -이 예외는, Java 2 SDK, Standard Edition, v1. 2 에서는 추천 되고 있지 않습니다.

인터페이스 java.rmi.registry.RegistryHandler 를 구현하는 클래스를 위한 패키지 접두사. 디폴트 값는 sun.rmi.server

java.rmi.server.packagePrefix

주 -이 예외는, Java 2 SDK, Standard Edition, v1. 2 에서는 추천 되고 있지 않습니다.

서버 패키지의 접두사. 전제로서 서버 참조 클래스의 구현 (UnicastRef 나 UnicastServerRef)은 접두사로 지정되는 패키지에 놓여진다. 디폴트 값는 sun.rmi.server
