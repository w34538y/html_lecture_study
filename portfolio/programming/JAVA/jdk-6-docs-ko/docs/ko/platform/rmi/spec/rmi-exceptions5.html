A. 3.1 java.rmi.ServerException 가 발생하는 원인

서버가 원격 메소드 호출을 실행중에 하층에 있어서의 예외가 서버상에 발생하는 일이 있습니다. 이러한 예외는 java.rmi.ServerException 에 랩 됩니다. 즉 java.rmi.ServerException 가 오리지날의 예외를 포함하기 (위해)때문에, 클라이언트는 그것을 추출할 수 있습니다. 이러한 예외는 ServerException 에 의해 랩 되어 클라이언트는 서버상의 원격 메소드에 대한 자신의 호출이 실패의 원인은 아니고, 서버의 2차 호출이 실패한 것을 알 수 있습니다.

예외
내용

java.rmi.server.SkeletonMismatchException

주 -이 예외는, Java 2 SDK, Standard Edition, v1. 2 에서는 추천 되고 있지 않습니다.

Stub와 스켈리턴의 해시 불일치

java.rmi.UnmarshalException

호출 헤더의 비정렬화중의 입출력 에러. 인수의 비정렬화중의 입출력 에러. 메소드 번호 또는 메소드 해시가 무효

java.rmi.MarshalException

반환값을 정렬화중의 프로토콜 에러

java.rmi.RemoteException

RemoteException 는 서버로부터의 원격 호출의 결과로서 발생한다
