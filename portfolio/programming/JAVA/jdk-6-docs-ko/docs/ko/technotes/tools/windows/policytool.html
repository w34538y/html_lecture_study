policytool - PolicyTool 관리 GUI 유틸리티

policytool 는, 유틸리티 GUI 경유로 취득한 사용자 입력에 근거해, plaintext의 정책 파일을 읽고 쓰기합니다.
  • 형식
  • 설명
  • 옵션
  • 관련 항목

형식

 
policytool 관리자 유틸리티를 실행합니다
policytool
policytool 를 실행해, 지정된 정책 파일을 읽어들입니다
policytool[-file][filename]
 
설명
file
로컬 정책 파일을 읽어들이도록(듯이) policytool 에 지시합니다
filename
파일명

설명

policytool 는, 시스템 관리 책임자가 로컬 정책 파일의 내용을 관리하는 것을 가능하게 하는 관리자용의 GUI 입니다. 정책 파일은 . policy 를 확장자(extension)에 가지는 plaintext 파일이며, 도메인별의 원격 요구자를 액세스권 객체에 매핑 합니다. 상세한 것에 대하여는,「Default Policy Implementation and Syntax」를 참조해 주세요.

옵션

file
filename 를 읽어들입니다.
