생성자 의 개요
jmap (map)
            java.util.Map 인스턴스를 취급하는데 편리한 스크립트 객체를 작성합니다.
 

생성자 의 상세

jmap

jmap(map)
    java.util.Map 인스턴스를 취급하는데 편리한 스크립트 객체를 작성합니다. 결과적으로 생기는 스크립트 객체의 필드명은, Map 의 열쇠입니다. 예를 들어, scriptObj.keyName 를 사용해, 지정한 열쇠와 관련지을 수 있었던 값에 액세스 할 수 있습니다.
    례: 
     
     var x = java.lang.SystemProperties();
     var y = jmap(x);
     print(y['java.class.path']); // prints java.class.path System property
     delete y['java.class.path']; // remove java.class.path System property
    파라미터:
      map - 랩 되는 java.util.Map 인스턴스
