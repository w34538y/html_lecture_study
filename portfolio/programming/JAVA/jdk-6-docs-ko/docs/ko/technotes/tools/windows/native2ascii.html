native2ascii - 네이티브 - ASCII 컨버터

native code (Latin 1 및 Unicode 이외)의 파일을 Unicode 코드로 변환합니다.

형식

native2ascii [options] [inputfile [outputfile]]

설명

Java 컴파일러 및 그 외의 Java 툴은, Latin-1 또는 Unicode 코드 (\udddd 표기) 문자를 포함한 파일만을 처리할 수가 있습니다. native2ascii 는, 이것들 이외의 문자 코드의 파일을 Latin-1 또는 Unicode 코드의 파일로 변환합니다.

outputfile 를 생략 했을 경우, 표준 출력에 출력됩니다. 게다가inputfile 를 생략 했을 경우, 표준 입력으로부터 입력됩니다.

옵션

-reverse
역의 처리를 실시합니다. 즉, Latin-1 또는 Unicode 코드의 파일을 native code로 변환합니다.

-encoding encoding_name
변환 처리로 사용하는 코드의 이름을 지정합니다. 디폴트의 코드는, 시스템 프로퍼티의 file.encoding 로부터 취득됩니다. encoding_name 캐릭터 라인은,「지원되고 있는 인코딩」 문서에 나타나고 있는, 지원되는 인코딩의 겉(표)의 최초의 열로부터 취득될 필요가 있습니다.
-Joption
Java 가상 머신에 option 를 건네줍니다. option 에는,「Java 어플리케이션 기동 툴」의 레퍼런스 페이지에 기재되어 있는 옵션을 1 개 지정합니다. 예를 들어,-J-Xms48m 와 지정하면(자), 스타트 업 메모리는 48M 바이트로 설정됩니다.
