jrunscript JavaScript 편입 함수, 편입 생성자

jrunscript 는, 사용자 스크립트를 평가하기 전에 짜넣어 함수 및 짜넣어 생성자 으로 스크립트 엔진을 초기화합니다. 이것에는, 단순한 파일 시스템 유틸리티 함수, 처리 유틸리티 (exec, exit 등), 몇개의 XML 유틸리티 및 단순한 입출력 함수의 몇개인가가 포함됩니다.
