메소드의 상세

cat

<static> void cat(obj, pattern)
    파일, URL 또는 InputStream 의 내용을 표시합니다.
    례: 
     
    cat('test.txt'); // show test.txt file contents
    cat('http://java.net'); // show the contents from the URL http://java.net
    파라미터:
      obj - 표시하는 입력 데이터
      pattern - 생략 가능. 패턴에 일치하는 행만을 표시

cd

<static> void cd(target)
    현재의 작업 디렉토리를 지정한 디렉토리로 변경합니다.
    파라미터:
      target - 변경하는 앞의 디렉토리. 생략 가능해, 디폴트는 사용자의 HOME

cp

<static> void cp(from, to)
    파일, URL, 또는 스트림을 다른 파일 또는 스트림에 카피합니다.
    파라미터:
      from - 입력 파일, URL 또는 스트림
      to - 출력 스트림 또는 파일

date

<static> void date()
    현재의 로케일을 사용해 현재의 일자를 출력합니다.

del

<static> void del(pathname)
    「rm」라고 동의입니다.

dir

<static> void dir(d, filter)
    「ls」라고 동의입니다.

dirname

<static> Object dirname(pathname)
    파일명의 디렉토리 부분을 돌려줍니다.
    파라미터:
      pathname - 입력 경로명
    반환값:
      지정된 파일명의 디렉토리 부분

echo

<static> void echo(x)
    지정한 캐릭터 라인 인수를 표시합니다.

exec

<static> void exec(cmd)
    child process를 실행해, 완료하는 것을 기다리고 나서 종료 코드를 돌려줍니다.
    파라미터:
      cmd - child process로 실행하는 커멘드

exit

<static> void exit(exitCode)
    쉘 프로그램을 종료합니다.
    파라미터:
      exitCode - OS 쉘에 반환되는 정수 코드. 생략 가능하고, 디폴트는 0

find

<static> void find(dir, pattern, callback)
    파일을 검색합니다. 일치하는 각각의 파일에 대해, 지정된 콜백 함수를 호출합니다.
    례: 
     
    find('. ') 
    find('. ', '. *\. class', rm);  // remove all . class files 
    find('. ', '. *\. java');       // print fullpath of each . java file 
    find('. ', '. *\. java', cat);  // print all . java files
    파라미터:
      dir - 검색하는 파일의 디렉토리
      pattern - 검색하는 패턴
      callback - 일치하는 파일에 대해서 호출하는 함수

grep

<static> void grep(pattern, files )
    Unix 의 grep 와 닮아 있습니다만, JavaScript 정규 표현 패턴을 인식합니다.
    파라미터:
      pattern - 검색하는 패턴
      files - 검색하는 1 개(살) 이상의 파일

ip

<static> void ip(name)
    지정한 도메인명의 IP 주소를 출력합니다.
    파라미터:
      name - 도메인명

load

<static> void load(str)
    JavaScript 코드를 스트림, 파일 또는 URL 로부터 로드해 평가합니다.
    례: 
     
    load('test.js'); // load script file 'test.js'
    load('http://java.sun.com/foo.js'); // load from a URL
    파라미터:
      str - 스크립트가 로드 되고 평가되는 입력원

ls

<static> void ls(dir, filter)
    디렉토리내의 파일을 일람표 가리킵니다.
    파라미터:
      dir - 파일을 리스트 표시하는 디렉토리. 생략 가능하고, 디폴트는 pwd
      filter - 리스트 표시되는 파일을 필터 하는 패턴. 생략 가능하고, 디폴트는 「.」

mkdir

<static> void mkdir(dir)
    새로운 디렉토리를 작성합니다.
    파라미터:
      dir - 새로운 디렉토리의 이름

mkdirs

<static> void mkdirs(dir)
    필요하지만 존재하지 않는 부모 디렉토리를 포함해 디렉토리를 작성합니다.
    파라미터:
      dir - 작성하는 디렉토리의 경로명

mv

<static> void mv(from, to)
    파일을 다른 장소로 이동합니다.
    파라미터:
      from - 파일의 전의 이름
      to - 파일의 새로운 이름

printf

<static> Object printf(format, args)
    C 를 닮은 printf 입니다.
    파라미터:
      format - 나머지의 출력 항목을 포맷 하는 캐릭터 라인
      args - 가변 인수의 리스트

pwd

<static> void pwd()
    작업 디렉토리를 출력합니다.

quit

<static> void quit(code)
    「exit」라고 동의입니다.

read

<static> Object read(prompt, multiline)
    prompt를 출력한 뒤, 표준 입력으로부터 1 행 또는 복수행을 읽어냅니다.
    파라미터:
      prompt - prompt. 생략 가능하고, 디폴트는 「>」
      multiline - 0 의 경우 단일행을 읽어내, 그 이외의 경우는 복수행을 읽어낸다

ren

<static> void ren(from, to)
    「mv」라고 동의입니다.

rm

<static> Object rm(pathname)
    지정한 파일을 삭제합니다.
    파라미터:
      pathname - 파일의 이름

rmdir

<static> void rmdir(pathname)
    지정한 디렉토리를 삭제합니다.
    파라미터:
      pathname - 디렉토리의 이름

which

<static> void which(cmd)
    PATH 에 있는 커멘드의 경로명을 출력합니다.
    파라미터:
      cmd - PATH 로부터 검색하는 커멘드의 이름

XMLDocument

<static> Object XMLDocument(inp)
    입력을 DOM 문서 객체로 변환합니다.
    파라미터:
      inp - 파일 또는 리더. 생략 가능하고, 없는 경우는 새로운 DOM 문서가 반환된다
    반환값:
      DOM 문서 객체를 돌려준다

XMLResult

<static> Object XMLResult(out)
    임의의 스트림, 파일을 XMLResult 로 변환합니다.
    파라미터:
      inp - 입력 스트림 또는 파일
    반환값:
      XMLResult 객체

XMLSource

<static> Object XMLSource(inp)
    임의의 스트림, 파일, 또는 URL 를 XMLSource 로 변환합니다.
    파라미터:
      inp - 입력 스트림, 파일, 또는 URL
    반환값:
      XMLSource 객체

XSLTransform

<static> void XSLTransform(inp, style, out)
    XSLT 변환을 실행합니다.
    파라미터:
      inp - 변환하는 입력 XML (URL, 파일 또는 InputStream)
      style - 사용하는 XSL Stylesheet (URL, 파일 또는 InputStream). 생략 가능
      out - 출력 XML (파일 또는 OutputStream)
