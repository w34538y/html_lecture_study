klist - Kerberos 에서의 자격 캐쉬 및 키 탭에 있어서의 엔트리의 표시

klist 를 사용하면, 로컬의 자격 캐쉬 및 열쇠 테이블의 엔트리를 표시할 수 있습니다.

형식

klist [ commands ]

설명

klist 는, 로컬의 자격 캐쉬 및 열쇠 테이블의 엔트리를 표시합니다. 사용자가,kinit 를 사용해 자격 캐쉬를 변경했는지,ktab 를 사용해 키 탭을 변경했을 경우, 이 변경을 확인하는 유일한 방법으로서klist 를 사용해 자격 캐쉬 또는 키 탭, 혹은 그 양쪽 모두의 내용을 표시할 수가 있습니다. klist 를 실행해도, Kerberos 데이타베이스는 변경되지 않습니다.

커멘드

사용 방법:klist [-c [-fe]] [-k [-tK]] [<<name>>] [-help]

커멘드 옵션 설명
-c 자격 캐쉬 엔트리를 일람표시 하도록(듯이) 지정한다
-k 키 탭 엔트리를 일람표시 하도록(듯이) 지정한다
<name> 자격 캐쉬명 또는 키 탭명을 지정한다. 이름이 지정되어 있지 않은 경우, 디폴트의 캐쉬명 및 키 탭이 사용된다. 디폴트의 캐쉬 및 키 탭은,kinit 의 메뉴얼 페이지로 지정되고 있다
-help 설명을 표시한다
-c 옵션이 지정되고 있는 경우 (자격 캐쉬의 옵션)
 -f 자격 플래그를 표시한다
-e 암호화의 종류를 표시한다
-k 옵션이 지정되고 있는 경우 (키 탭의 옵션)
 -t 키 탭 엔트리의 타임 스탬프를 표시한다
-K 키 탭 엔트리의 DES 키를 표시한다

례

키 탭 엔트리의 타임 스탬프와 DES 키를 포함해, 열쇠 테이블내의 엔트리를 일람표시 하는 경우
klist -k -t -K FILE:\temp\mykrb5cc
 
자격 플래그와 주소 리스트를 포함해, 자격 캐쉬의 엔트리를 일람표시 하는 경우
klist -c -f FILE:\temp\mykrb5cc
