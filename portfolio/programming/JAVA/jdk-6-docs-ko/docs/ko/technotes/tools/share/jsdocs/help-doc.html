API 문서의 구성

이 API (Application Programming Interface) 문서에는, 다음에 설명하는 네비게이션 바에 있는 항목에 대응하는 페이지가 포함됩니다.

클래스

각 클래스에는 그것 자신의 개별의 페이지가 있습니다. 이러한 각 페이지에는 다음과 같이, 클래스의 설명, 개요 테이블, 및 멤버의 상세 설명으로 구성되는 3 개의 섹션이 포함되어 있습니다:

  • 클래스 계층 표시
  • 직계의 서브 클래스
  • 클래스의 선언
  • 클래스의 설명

  • 필드의 개요
  • 생성자 의 개요
  • 메소드의 개요

  • 필드의 상세
  • 생성자 의 상세
  • 메소드의 상세
각 개요 엔트리에는, 그 항목의 상세한 기술중에서 1 행 째의 문장이 표시됩니다.

색인

색인 에는, 모든 클래스, 생성자 , 메소드, 및 필드의 알파벳순서의 리스트가 포함됩니다.

전/차

이러한 링크에 의해, 전 또는 다음의 클래스, 인터페이스, 패키지, 또는 관련 페이지로 이동할 수 있습니다.

프레임 있어/프레임 없음

이러한 링크는 HTML 프레임을 표시하거나 숨기거나 합니다. 모든 페이지는 프레임에서도, 프레임없이도 표시할 수 있습니다.

이 헬프 파일은, 표준 doclet 를 사용해 생성된 API 문서에 적용됩니다.
