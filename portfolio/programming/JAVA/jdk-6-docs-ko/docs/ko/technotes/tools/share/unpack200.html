형식

unpack200 [ options ] input-file JAR-file

옵션의 지정 순서에 결정은 없습니다. 커멘드행으로 지정된 마지막 옵션이, 그 이전으로 지정된 모든 옵션에서 우선됩니다.

input-file
입력 파일의 이름. 입력 파일은 일반적으로, pack200 gzip 파일이나 pack200 파일입니다. 이 밖에, 0 을 설정하면 pack200(1)로부터 작성된 JAR 파일도 입력 파일로서 사용할 수 있습니다. 이 경우, 입력 파일의 내용이 Pack200 마커와 함께 출력 JAR 파일에 카피됩니다.
JAR-file
출력 JAR 파일명.

설명

unpack200 는,pack200(1)로 작성된 팩 파일을 JAR 파일로 변환하는 네이티브 구현입니다. 일반적인 사용 방법:

% unpack200 myarchive.pack.gz myarchive.jar

이 예에서는, 디폴트의 unpack200 설정으로,myarchive.jar 로부터 myarchive.pack.gz 가 작성됩니다.

옵션

표준 옵션

-Hvalue --deflate-hint=value

JAR 파일내의 모든 엔트리에 true,false,keep 로서 디플레이션을 설정합니다. 디폴트 모드는 keep 입니다. true 또는 false 의 경우, 디폴트의 동작을 오버라이드(override) 해, 출력 JAR 파일내의 모든 엔트리의 디플레이션 모드를 설정합니다.

비표준 옵션

-r --remove-pack-file

입력 팩 파일을 삭제합니다.

-v --verbose

최소한의 메세지를 출력합니다. 이 옵션을 복수 지정하면(자), 보다 긴 메세지가 출력됩니다.

-q --quiet

메세지를 표시하지 않고 동작합니다.

-lfilename --log-file=filename

출력 메세지의 로그 파일을 지정합니다.
