애플릿과 어플리케이션의
데먼스트레이션

 문서의 목차

    애플릿

    애플릿과 원시 코드의 데먼스트레이션은, 다음의 장소로부터 입수할 수 있습니다.

    • java.sun.com Web 사이트의애플릿과 원시 코드의 데모

    • JDK 의 인스톨처의 demo/applets 디렉토리에 놓여진 애플릿과 원시 코드의 데모

    애플릿을 실행하려면 , 브라우저로 그 HTML 페이지를 열어, 애플릿 링크를 클릭하는지, AppletViewer 로 HTML 페이지를 엽니다.

    브라우저로 애플릿을 실행하려면 , JavaTM Plug-in 가 필요합니다. 이것은 디폴트로 JDK 및 JRE 의 인스톨시에 인스톨 되고 있습니다. AppletViewer 는, JDK 의 bin\appletviewer 로 제공되고 있습니다. AppletViewer 및 그 외의 JDK 툴의 사용에 대한 자세한 것은,「JDK 툴과 유틸리티」를 참조해 주세요.

    어플리케이션

    어플리케이션과 샘플 코드도, JDK 의 인스톨처로부터 이용 가능합니다. 이것들은, Java 2 플랫폼의 다양한 기능을 실제로 가리킵니다. 이용할 수 있는 데모에 대해, 지금부터 설명합니다.

    JFC 어플리케이션과 애플릿

    JFC 어플리케이션과 애플릿은 JDK 의 demo/jfc 디렉토리에 있어, Swing 및 Java 2DTM 테크놀러지를 실제로 가리킵니다.

    Java Platform Debugger Architecture

    JDK 의 demo/jpda/examples.jar 파일에는, jdb Java 디버거의 원시 코드 및 javadt GUI 디버거의 prototype 등, JPDA 의 예가 포함되고 있습니다.

    Poller Demo (SolarisTM 오퍼레이팅(operating) 환경의 경우만)

    이 데모는 JDK 의 demo/jni/Poller 디렉토리에 있어, Java 프로그램 언어에 의한 입출력 다중화 기능에의 액세스를 실시합니다. Solaris poll 의 메뉴얼 페이지를 참조해 주세요.

    추가의 어플리케이션 및 애플릿

    추가의 애플릿 및 어플리케이션은, 다음의 장소에서 입수할 수 있습니다.
    • 「기능 가이드」 - 특정의 기능의 문서에 샘플이 포함되어 있습니다.
    • 「Applets 페이지」 - Sun 의 Java Software Web 사이트에 있습니다.
    • 「Java 튜토리얼」 - Sun 의 Java Software Web 사이트에 있습니다.
