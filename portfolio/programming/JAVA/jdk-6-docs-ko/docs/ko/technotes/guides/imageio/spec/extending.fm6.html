4.6 스트림 플러그 인의 작성

javax.imageio.stream 패키지에 포함되는 표준 클래스를 이용하면(자), 다음과 같은 기능을 가지는 ImageInputStream 나 ImageOutputStream 를 작성할 수 있습니다.

  • File 로부터 읽어들인다
  • 일시 File 를 캐쉬로서 사용해,InputStream 로부터 읽어들인다
  • 메모리내의 byte 배열을 캐쉬로서 사용해,InputStream 로부터 읽어들인다
  • File 에 기입한다
  • 일시 File 를 캐쉬로서 사용해,OutputStream 에 기입한다
  • 메모리내의 byte 배열을 캐쉬로서 사용해,OutputStream 에 기입한다
또, 다른 입출력 프리미티브(primitive) (예를 들어, 데이타베이스나, 장래의 고속의 입출력 인터페이스등)과의 인터페이스가 되는 ImageInputStream 및 ImageOutputStream 의 새로운 구현을 작성하는 일도 가능합니다. ImageInputStreamImpl 및 ImageOutputStreamImpl 클래스는, 서브 클래스화를 위한 기초가 됩니다. 서브 클래스에서 구현할 필요가 있는 메소드 (아래와 같이)는 그만큼 많지 않습니다가, 처리의 효율화를 위해서(때문에) 한층 더 많은 메소드를 구현할 수도 있습니다.

  • int read()
  • int read(byte[] b, int off, int len)
  • write(int b)
  • write(byte[] b, int off, int len)
  • seek(long pos)
