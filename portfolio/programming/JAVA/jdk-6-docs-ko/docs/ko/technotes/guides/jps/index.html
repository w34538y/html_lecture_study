Java 인쇄 서비스

 문서의 목차
Java 인쇄 서비스 API 는 버젼 1.4 로 도입된 API 로, J2ME 프로파일등의 사이즈가 제한되는 플랫폼을 포함해 모든 Java 플랫폼상에서 인쇄 기능을 제공합니다. 또, J2SE 버젼 1.2 로 도입된 java.awt.print.PrinterJob API 도 지원합니다. Java 인쇄 서비스의 API 에는, IETF 의 IPP (Internet Printing Protocol) 1.1 에 규정되고 있는 표준 속성에 근거한, 확장 가능한 인쇄 속성 세트가 짜넣어지고 있습니다. 클라이언트 및 서버의 어플리케이션은, 이러한 속성을 사용해, 속성으로 지정된 기능을 가지는 프린터를 검출 및 선택할 수가 있습니다. 새롭게 추가된 StreamPrintService 를 사용하면(자), 데이터 서식을 어플리케이션으로 변환할 수 있습니다. 또, 서비스 프로바이더 인터페이스를 사용하면, Sun 이외의 벤더가 독자적인 인쇄 서비스를 동적으로 인스톨 할 수 있습니다.

API 의 스펙

Java 인쇄 서비스 API 는, 다음의 4 개의 패키지로부터 구성됩니다.

javax.print -
Java 인쇄 서비스 API 의 주체가 되는 클래스와 인터페이스를 제공합니다.

javax.print.attribute -
Java 인쇄 서비스 API 의 속성의 종류 및 그것들을 수집해 속성 세트에 정리하는 방법을 기술한 클래스와 인터페이스를 제공합니다.

javax.print.attribute.standard -
특정의 인쇄 속성을 정의하는 클래스를 제공합니다.

javax.print.event -
인쇄 서비스 및 특정의 인쇄 작업의 진척을 감시하는 클래스와 청취자 인터페이스를 제공합니다.

튜토리얼 및 개발자 안내

「Java 인쇄 서비스 API 사용자 가이드」 (HTML (일본어판) | PDF (영문판) )에서는, Java 인쇄 서비스에 관한 상세와 Java 인쇄 서비스 API 를 사용해 이하의 조작을 실시하는 방법에 대해 설명합니다.
  • 인쇄 서비스 기능에 근거해, 인쇄 서비스를 검출 및 선택한다
  • 인쇄 데이터의 형식을 지정한다
  • 인쇄하는 문서가 지원되는 서비스에 인쇄 작업을 송신한다
