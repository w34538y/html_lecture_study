드래그＆드롭

 문서의 목차
드래그＆드롭에 의해, JavaTM 프로그램 언어와 네이티브 어플리케이션의 사이, 다른 Java 프로그램 언어 어플리케이션간, 및 동일한 Java 언어 어플리케이션내에서의 데이터 전송이 가능합니다.

드래그＆드롭의 지원가 AWT 패키지의 주된 구현입니다만, Swing GUI 클래스에 추가의 지원가 짜넣어지고 있습니다. 많은 경우, Swing 컴퍼넌트의 메소드의 1 개를 호출하는 것으로 드래그＆드롭의 지원를 유효하게 할 수 있습니다. 상세한 것에 대하여는, 「The Swing Tutorial 」의 「How to Use Drag and Drop and Data Transfer 」를 참조해 주세요.
