JMX (Java Management Extensions)의 트레이스

 문서의 목차

JMX 구현의 트레이스는, Java SE 로그 기능 (java.util.logging 패키지내의 클래스에 의해 제공)에 근거하고 있습니다. JMX 구현에서는, 로거세트를 사용해 트레이스를 송신합니다. 모든 로거는, 모든 public JMX 인터페이스를 호스트 하는 루트 패킷명에 대응한 javax.management 이름 공간에 포함됩니다.

다음의 리스트는, 구현에 사용되는 로거객체와 거기에 대응하는 동작 카테고리를 나타내고 있습니다.

로거이름: 로그 기록되는 정보
javax.management.mbeanserver MBean 서버에 관한 정보
javax.management.mlet MLet 서비스로부터 얻을 수 있는 정보
javax.management.monitor 모니터로부터 얻을 수 있는 정보
javax.management.timer 타이머부터 얻을 수 있는 정보
javax.management.notification 통지 기구로부터 얻을 수 있는 정보
javax.management.relation 관계 서비스로부터 얻을 수 있는 정보
javax.management.modelmbean Model MBean 컴퍼넌트로부터 얻을 수 있는 정보
javax.management.misc 다른 클래스로부터 보내지는 정보
javax.management.remote.rmi RMI 연결기로부터 얻을 수 있는 정보
javax.management.remote.misc 다른 JMX Remote API 클래스로부터 보내지는 정보

JMX 에서는 다음의 로그 레벨이 사용됩니다.

JMX 의 로그 레벨: 설명
Level.FINEST 디버그 트레이스의 출력에 사용 (JMX 구현의 진단에 유효)
Level.FINER 정보 트레이스의 출력에 사용
Level.SEVERE 에러 상태의 통지에 사용
