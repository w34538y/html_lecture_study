HTML 컨버터를 사용한 Java Plug-in 용 Applet 태그의 변환


Java Plug-in HTML 컨버터는, 애플릿을 포함한 HTML 페이지 (파일)를 Java Plug-in 의 형식으로 변환하는 유틸리티입니다. 변환은, 다음의 방법으로 실시합니다.
 
최초로, 애플릿의 일부에서는 없는 HTML 가, 원시 파일로부터 임시 파일에 전송 됩니다. 다음에,<APPLET> 태그에 이른 시점에서, 컨버터는 최초의 </APPLET> 태그 (인용부호로 둘러싸이지 않았다)까지를 구문 분석 해, 애플릿 데이터를 템플릿과 머지 합니다 (자세한 것은,「HTML 컨버터의 상세」라고 하는 섹션의「템플릿의 상세」를 참조). 에러가 발생하지 않고 이 조작이 완료했을 경우, 원의 HTML 파일이 백업 폴더에 이동되어 임시 파일에 원의 파일명을 붙일 수 있습니다.  

컨버터는, 배치된 파일을 효과적으로 변환합니다. 이 때문에, 컨버터를 일단 실행하면(자), 파일이 Java Plug-in 용으로 설정됩니다.

HTML 컨버터의 인스톨 및 사용 방법의 자세한 것은,「HTML 컨버터의 상세」를 참조해 주세요.
