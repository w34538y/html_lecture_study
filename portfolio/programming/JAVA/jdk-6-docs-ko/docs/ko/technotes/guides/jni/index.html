Java Native Interface

 문서의 목차
JavaTM Native Interface (JNI) 는, Java 네이티브 메소드를 쓰거나 Java 가상 머신* 을 네이티브 어플리케이션에 짜넣거나하기 위한 표준 프로그래밍 인터페이스입니다. 주된 목적은, 주어진(given) 플랫폼상의 모든 Java 가상 머신의 구현간으로, 네이티브 메소드 라이브러리를 binary level compatibility로 하는 것에 있습니다.
