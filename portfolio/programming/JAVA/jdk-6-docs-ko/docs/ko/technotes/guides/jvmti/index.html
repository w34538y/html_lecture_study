JavaTM Virtual Machine Tool Interface (JVM TI)

문서의 목차

JVMTM Tool Interface (JVM TI)는, 툴용의 새로운 네이티브 프로그래밍 인터페이스입니다. JVM TI 는, JavaTM 가상 머신 (JVM)으로 동작하는 어플리케이션 상태 검사와 실행 제어의 양쪽 모두의 기능을 제공해, 프로 파일링 툴, 디버그 툴, 감시 툴, thread 분석 툴, 커버리지(coverage) 분석 툴 등, JVM 상태 그 외에 액세스 할 필요가 있는 각종 툴을 지원합니다.

주: JVM TI 는 JDKTM 5.0 으로 도입되었습니다. JVM TI 는 Java Virtual Machine Profiler Interface (JVMPI)와 Java Virtual Machine Debug Interface (JVMDI)에 대신하는 것입니다. JVMPI 와 JVMDI 는 JDK 6 에서는 제공되지 않습니다.

JVM TI 레퍼런스

  • 개요, API 레퍼런스, 및 사용법에 대해서는,「JVM TI 레퍼런스」 를 참조해 주세요.

API 의 확장 기능

  • JDK 6 의 API 의 확장 기능 - 릴리스 5.0 에서 6 에의 이행에 수반하는 API 의 변경점의 주석 첨부 리스트.

상세 정보

  • Java 가상 머신 - 기능, 커멘드행 옵션, 추가 정보.
  • Java Native Interface - JVM TI 가 확장하는 인터페이스.
  • java.lang.instrument 패키지 - JVM TI 의 Instrumentation 기능만을 사용하는 경우에 JVM TI 에 대신하는 Java 프로그램 언어.

*이 Web 사이트에서는, 「Java 가상 머신」또는 「JVM」라고 하는 용어는, Java 플랫폼용의 가상 머신을 나타냅니다.
