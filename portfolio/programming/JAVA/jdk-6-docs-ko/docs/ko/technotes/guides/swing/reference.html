Swing 패키지

 문서의 목차

  • javax.swing - 일련의 「경량」(native code를 사용하지 않는 Java 로 기술되고 있다) 컴퍼넌트를 제공합니다. 이 컴퍼넌트는, 가능한 한 모든 플랫폼에서 똑같이 동작합니다.
  • javax.swing.border - Swing 컴퍼넌트의 주위에 특수한 경계선을 draw하기 위한 클래스와 인터페이스를 제공합니다.
  • javax.swing.colorchooser - JColorChooser 컴퍼넌트가 사용하는 클래스와 인터페이스를 포함합니다.
  • javax.swing.event - Swing 컴퍼넌트가 트리거하는 이벤트를 지원합니다.
  • javax.swing.filechooser - JFileChooser 컴퍼넌트가 사용하는 클래스와 인터페이스를 포함합니다.
  • javax.swing.plaf - 플러그 인 가능한 Look & Feel 기능을 제공하기 위해서 Swing 가 사용하는 1 개의 인터페이스 및 많은 추상 클래스를 제공합니다.
  • javax.swing.plaf.basic - Basic Look ＆ Feel 에 근거해 구축된 사용자 인터페이스 객체를 제공합니다.
  • javax.swing.plaf.metal - 디폴트의 Look & Feel 인 Java Look & Feel (이전의 코드명 Metal)에 근거해 구축된 사용자 인터페이스 객체를 제공합니다.
  • javax.swing.plaf.multi - 2 개(살) 이상의 Look ＆ Feel 를 결합하는 사용자 인터페이스 객체를 제공합니다.
  • javax.swing.plaf.synth - 모든 페인트가 위양 되는 스킨 설정 가능한 Look ＆ Feel 의 사용자 인터페이스 객체를 제공합니다.
  • javax.swing.table - JTable 를 처리하기 위한 클래스와 인터페이스를 제공합니다.
  • javax.swing.text - 편집 가능 및 편집 불가능한 텍스트 컴퍼넌트를 처리하기 위한 클래스와 인터페이스를 제공합니다.
  • javax.swing.text.html - HTML 텍스트 문자 편집기를 작성하기 위한 클래스 HTMLEditorKit 와 지원 클래스를 제공합니다.
  • javax.swing.text.html.parser - 지원 클래스와 함께 디폴트의 HTML 퍼서를 제공합니다.
  • javax.swing.text.rtf - Rich Text Format 텍스트 문자 편집기를 작성하기 위한 클래스 (RTFEditorKit)를 제공합니다.
  • javax.swing.tree - JTree 를 처리하기 위한 클래스와 인터페이스를 제공합니다.
  • javax.swing.undo - 개발자는 텍스트 문자 편집기등의 어플리케이션으로 바탕으로 되돌리는/재시도를 지원할 수 있게 됩니다.
