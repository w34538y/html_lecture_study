1.2 이 문서에 대해

이 문서는, API 에 대해 간결하게 설명한 가이드로, 간단한 프로그래밍예도 게재되고 있습니다. 그러나, 이 문서는, API 의 완전한 가이드로서의 이용을 의도한 것이 아닙니다. 스펙에 대한 완전한 가이드는, 자동 생성된 ("javadoc") API 문서입니다. 이 문서에 기재되어 있는 설명 자료보다, 자동 생성된 문서가, 언제라도 우선됩니다.
