미국 Sun Microsystems, Inc. (이하 「산」이라고 한다)(은)는 여기에, Java Image I/O API 시방서 (이하 「본시방서」라고 한다)에 있어서의 산의 지적 소유권에 근거해, 본시방서를 내부 평가의 목적에 한해서 사용하는, 비독점적, 양도 금지, 한편 전세계에서 유효한 한정 라이센스 (서브 라이센스를 줄 권리는 포함하지 않는다)를 무상으로 손님에게 허락합니다. 이 한정 라이센스를 따로 해, 본시방서에 관련되는 어떠한 권리, 권한, 또는 권익을 손님이 취득할 것은 없고, 본시방서를 생산 용도 또는 상업 용도에 사용할 권리를 손님이 가질 것은 없습니다.

본시방서로 공개되는 내용은, 산의 소유에 속하는 비밀 정보 (이하 「비밀 정보」라고 한다)로 구성됩니다. 손님은, 비밀 정보를 제삼자에게 개시해 안되어, 본라이센스의 조항에 근거해서만, 그것들을 사용하는 것으로 합니다.

본서는 「현상인 채」제공되어 상품성, 특정 목적 적합성 또는 제삼자의 권리에 관한 비침해의 묵시의 보증을 포함하지만 이것으로 한정되지 않는, 명시적일까 묵시적인가를 묻지 않는, 어떠한 보증도 행해지지 않는 것으로 합니다. 산은, 본시방서의 사용에 기인해 손님이 입은 어떠한 손해에 대해서도, 책임을 지지 않습니다.

본시방서로 공개된 내용에는, 기술적인 잘못해 또는 미스프린트가 있을 가능성이 있습니다. 한층 더 그 내용은 정기적으로 변경되어 걸리는 변경은 본시방서의 최신판에 반영됩니다. 또, 산은 본시방서의 내용을, 언제라도 그 단독의 최선에 의해 개량 또는 변경하는 일이 있습니다. 산은, 본시방서의 후계 버젼을 개발할 의무, 또는 본시방서에 근거하는 어떠한 제품이나 테크놀러지를 개발할 의무를 지지 않습니다. 또, 본시방서 또는 그 관련 테크놀러지를, 현재 또는 장래에 있어, 생산 그 외의 용도를 위해서(때문에) 라이센스 허락하는 어떠한 의무도, 산은 지지 않습니다.
