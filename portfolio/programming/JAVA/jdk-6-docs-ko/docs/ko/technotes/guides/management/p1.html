Java SE 감시 및 관리 가이드

처음에

Java Platform, Standard Edition (Java SE 플랫폼) 6 은, Java 가상 머신 (Java VM) 및 게다가로 실행되는 Java 어플리케이션의 퍼포먼스의 감시와 관리를 가능하게 하는 다양한 유틸리티를 갖추고 있습니다. 「Java SE 감시 및 관리 가이드」에서는, 그러한 감시 및 관리의 유틸리티에 대해 설명합니다.

대상으로 하는 독자

「Java SE 감시 및 관리 가이드」는, 시스템 관리 책임자나 소프트웨어 개발자 등, Java 언어의 경험을 가져, Java 플랫폼 및 거기서 사용되는 어플리케이션의 퍼포먼스를 매우 중요로 하는 사용자를 대상으로 하고 있습니다.

본서를 읽기 전에

Java SE 플랫폼이 가지는 다른 몇개의 기능에 익숙 하는 것을 추천합니다. 이 점으로써 도움이 되는 문서는 다음과 같습니다.

  • Java Management Extensions (JMX) 테크놀러지

  • Java HotSpot 테크놀러지

  • Java 가상 머신 테크놀러지

  • Java HotSpot VM 의 퍼포먼스 문서

본서의 구성

본가이드에서는, 다음의 토픽에 대해 설명합니다.

  • 제 1 장 「Java SE 감시 및 관리의 개요」 : Java SE 플랫폼에 의해 제공되는 감시 및 관리를 위한 유틸리티에 대해 설명합니다.

  • 제 2 장 「JMX 테크놀러지를 사용하는 감시와 관리」 : JMX API 를 사용해 감시와 관리를 행하기 위한 플랫폼의 구성 방법에 대해 설명합니다.

  • 제 3 장 「JConsole 의 사용」 : JConsole 그래피컬 사용자 인터페이스에 대해 설명합니다.

  • 제 4 장 「플랫폼 MBean 서버와 플랫폼 MXBean 의 사용」 : 감시 및 관리를 목적으로 해 Java SE 플랫폼에 준비된 MBean 에 대해 설명합니다.

  • 제 5 장 「SNMP 감시와 관리」 ：Simple Network Management Protocol (SNMP)를 사용한 감시 및 관리의 실시 방법에 대해 설명합니다.

  • 부록 A 「Microsoft Windows 용의 시큐리티 정보의 상세」 : Windows 플랫폼 특유의 시큐리티 구성 정보에 대해 설명합니다.

관련하는 것 외 회사의 Web 사이트 참조

본가이드에서는 타사의 URL 를 참조하고 있어, 이 URL 로부터 추가의 관련 정보를 취득할 수 있습니다.

주 - Sun 는, 본가이드에 게재된 타사의 Web 사이트를 이용할 수 있는지 아닌지에 임해서 책임을 지지 않습니다. Sun 는, 그러한 사이트 또는 자원으로부터 직접 또는 간접적으로 이용할 수 있는 어떠한 내용, 광고, 제품, 그 외의 물질에 대해서도 보증을 실시하는 것은 아니고, 또, 책임을 지는 것이기도 하지 않습니다. Sun 는, 그러한 사이트 또는 자원으로부터 직접 또는 간접적으로 이용할 수 있는 내용, 상품, 서비스등에 의해, 또는 그것들을 사용 혹은 신뢰했던 것에 관련해, 실제로 발생한 손해 또는 발생했다고 여겨지는 손해에 대해, 어떠한 책임도 지는 것이 아닙니다.
