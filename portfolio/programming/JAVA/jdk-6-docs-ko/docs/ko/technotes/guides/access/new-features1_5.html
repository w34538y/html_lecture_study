Accessibility

JavaTM 2 Platform Standard Edition 5 의 신기능

 Java Accessibility
여기서 설명하는 기능은, Java 2 Platform Standard Edition 5 로 도입되었습니다. 이러한 기능도, Java 플랫폼의 향후의 릴리스에 짜넣어집니다. 이러한 추가의 상당수는 http://developer.gnome.org/projects/gap 로 설명되도록(듯이), Java Accessibility API 나 GNOME Accessibility Framework 와의 호환성을 유지하기 위해서 행해졌습니다.

API 의 확장 기능
JavaTM Accessibility API 에 다음의 클래스 및 인터페이스가 추가되었습니다.
  • javax.accessibility.AccessibleExtendedText
  • javax.accessibility.AccessibleStreamable
AccessibleExtendedText
AccessibleExtendedText 인터페이스는, 점자 및 화면 리더 리뷰 모드를 지원하고 있습니다.
AccessibleStreamable
AccessibleStreamable 인터페이스는, MIME 타입에 근거하는 raw 데이터를, 스트림 인터페이스에 제공합니다.
JSpinner 및 ProgressMonitor 용의 Accessibility 지원
이 릴리스에서는, Swing 의 스피나 및 진척 모니터용의 accessibility 지원가 제공됩니다.
