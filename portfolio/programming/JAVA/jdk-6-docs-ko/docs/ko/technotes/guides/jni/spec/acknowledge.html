사례

Java Native Interface (JNI) 스펙은, JavaSoft 와 Java 의 라이센스 계약자와의 사이에 대화를 거듭한 결과적으로 작성되었습니다. 목표는, 네이티브 인터페이스 표준의 실현입니다.

JavaSoft 의 Sheng Liang 는, JNI 의 설계와 스펙을 담당하고 있습니다. JNI is partly evolved from Netscape’s Java Runtime Interface (JRI), which was designed by Warren Harris.Warren Harris 도 JNI 의 설계를 개량하기 위해서 협력해 주었습니다. IBM 의 Simon 와 Apple 의 Patrick Beard 는, 광범위하게 미치는 피드백을 제공해, 이것에 의해, 다양한 방법으로 JNI 설계가 형성되었습니다.

게다가 JNI 는 JavaSoft 사내의 설계 리뷰로부터도 혜택을 받았습니다. 이러한 설계 리뷰는, James Gosling, Peter Kessler, Tim Lindholm, Mark Reinhold, Derek White, 및 Frank Yellin 에 의해 행해졌습니다.

또, 많은 사용자로부터 받은, 다양한 스펙의 원안에 관한 유익한 코멘트나 제안에도 감사 드리겠습니다.

마지막으로, 이 문서의 표시를 개량해 준 Beth Stearns 에 감사 드리겠습니다.
