Preferences API

 문서의 목차
어플리케이션에는, 다양한 사용자 및 환경 요건에 응한 설정 및 구성 데이터가 필요합니다. java.util.prefs 패키지를 사용하면, 사용자 및 시스템의 설정 및 구성 데이터를, 어플리케이션으로부터 포함 및 취득할 수가 있습니다. 이 데이터는, 구현 마다 다른 배킹 스토어에 지속적으로 보존됩니다. 설정 노드는 2 개의 개별의 트리로 구성됩니다. 한편은 사용자 설정용, 이제(벌써) 한편은 시스템 설정용입니다.
