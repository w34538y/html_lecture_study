주석 처리 툴 (apt)

 문서의 목차
apt 는, 주석 처리를 위한 커멘드행 유틸리티입니다. 일련의 관련하는 API 와 지원하는 인프라스트럭쳐(infrastructure)에 의해,주석 (JSR 175 ) 프로그램을 처리합니다. 이러한 관련하는 API 에 의해, 프로그램 구조를, 빌드시에, 소스 베이스 또 읽어내 전용으로 보는 것이 가능하게 됩니다. 이것들은,총칭 (JSR 14 ) 프로그램의 추가 후에, JavaTM 프로그램 언어의 형태 시스템을 정상적으로 모델 하기 위해서 설계되었습니다.

apt 는, 최초로 새로운 원시 코드와 다른 파일을 작성하는 주석 프로세서를 실행합니다. 다음에,apt 는, 원의 파일과 작성된 원시 파일의 컴파일을 실시합니다. 이것에 의해 개발 사이클은 간소화됩니다.
