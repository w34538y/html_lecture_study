Minor changes to java.io in the JavaTM 2 Platform v1. 2

 JDKTM 목차
버젼 1.2 의 JavaTM 2 플랫폼에서는, 패키지 java.io 에 대해서 다음의 변경이 도입되었습니다. 큰 변경이 아닙니다.
  • 큰 변경이 아닙니다. PrintWriter 클래스의 out 필드가 private 로부터 protected 로 변경되었기 때문에, 서브 클래스로부터 액세스 할 수 있습니다.
  • The RandomAccessFile class supports a new setLength method; and
  • StringWriter(int) 생성자 이, public 가 되었습니다.

Java 2 플랫폼에서는, 다음의 생성자 에 의해, 입력 파라미터가 null 가 아닌지 어떤지가 검사됩니다.

  • 생성자 PrintStream(OutputStream out) 및 PrintStream(OutputStream out,  boolean autoFlush) 는,out 파라미터가 null 의 경우에 NullPointerException 를 throw 합니다.

  • 생성자 InputStreamReader(InputStream in) 및 InputStreamReader(InputStream in,  String enc) 는,in 파라미터가 null 의 경우에 NullPointerException 를 throw 합니다.

  • 생성자 OutputStreamWriter(OutputStream out) 및 OutputStreamWriter(OutputStream out,  String enc) 는,out 파라미터가 null 의 경우에 NullPointerException 를 throw 합니다.

  • protected 생성자 Reader(Object lock) 및 Writer(Object lock) 는,lock 파라미터가 null 의 경우에 NullPointerException 를 throw 하게 되었습니다.
