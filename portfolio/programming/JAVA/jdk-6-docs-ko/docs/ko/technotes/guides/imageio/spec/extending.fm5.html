4.5 변환 플러그 인의 작성

변환 플러그 인은,ImageTranscoderSpi (다른 플러그 인의 Spi 클래스와 같은 기능을 실행한다)와ImageTranscoder 인터페이스를 구현하는 객체로 구성됩니다. 

IIOMetadata convertStreamMetadata(IIOMetadata inData,
                                  ImageWriteParam param);

IIOMetadata convertImageMetadata(IIOMetadata inData,
                                 ImageTypeSpecifier imageType,
                                 ImageWriteParam param);
ImageTranscoder 에서는, 표준의 인터페이스를 사용해 입력 메타데이타를 unpack 하는 일도, 처리하고 있는 실제의 객체에 고유의 인터페이스를 이용할 수도 있습니다. 예를 들어, 이전의 예로 정의한 MyFormatMetadata 클래스의 keywords 및 values 인스턴스 변수에 액세스 할 수 있습니다. 이러한 변수가 공개되고 있는 (다만, 문서화되어 있지 않다) 것은, 확실히, 변환 플러그 인의 개발자가 DOM 표현을 더듬는 필요없음에 그러한 변수에 액세스 할 수 있도록(듯이) 하기 (위해)때문에입니다.
