부록 B:클래스의 그림의 범례

클래스의 그림에서는, Java 엔티티를 다음과 같은 기호로 구별합니다.

  1. 인터페이스:각의 둥근 구형
  2. 클래스:구형
  3. abstract 클래스:흰 환인 ()이 있는 구형
  4. final 클래스:검은 환인 ()이 있는 구형
  5. 서브 클래스가 있는 클래스:하 오른쪽 코너에 작은 검은 삼각형이 있는 구형

    이러한 원가요소의 일부를, 다음에 나타냅니다. 이 장으로 설명하는 클래스 또는 인터페이스에는, 회색의 그림자가 붙어 있습니다 (이것은, 패키지 클래스의 그림에는 없다). 실선은 extends 를, 점선은 implements 를 나타내고 있습니다.
