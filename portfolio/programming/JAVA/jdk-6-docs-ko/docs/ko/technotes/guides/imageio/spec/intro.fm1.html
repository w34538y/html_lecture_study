1.1 Java Image I/O API

Java Image I/O API 는, 파일에 포함된 이미지나, 네트워크 경유로 액세스 하는 이미지를 처리하기 위한 , 플러그 인 가능한 아키텍쳐(architecture)를 제공합니다. 이 API 는, 이미지를 로드하거나 보존하거나하기 위한 종래의 API 에 비해, 유연성이 꽤 높고, 기능도 강력합니다.

이 API 는, 이미지의 형식에 특유의 플러그 인을 추가하기 위한 시스템를 제공합니다. 자주(잘) 사용되는 몇개의 이미지 형식용의 플러그 인이, 표준의 Java SDK 에 짜넣어질 예정입니다. 그 외의 플러그 인도, Sun 나 서드 파티로부터 입수할 수 있게 되겠지요. Java 개발자가 어느 이미지 형식에 관심이 있는지를 표명하는 장소로서 투표 프로세스가 준비되었습니다. 이하를 참조해 주세요.
