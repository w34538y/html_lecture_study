커스텀 데이터 형식과 직렬화의 사용

예제의 실행

이 예제를 컴파일 해 실행하려면 , 다음의 순서에 따릅니다.

  1. 자신의 베이스 디렉토리아래의 objectserializationexamples/CustomDataFormat 디렉토리에서, JDK TM 1.1 이후를 사용해 CustomDataExample.java 를 컴파일 합니다. 
      javac CustomDataExample.java
  2. 컴파일 한 파일을 실행합니다. 
      java CustomDataExample

2 개의 배열이 출력됩니다.

  • 최초의 배열은 직렬화전의 배열입니다.
  • 2 번째의 배열은 직렬화 복원 후의 배열입니다.
