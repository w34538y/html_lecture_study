Java Naming and Directory Interface TM (JNDI)의 개요
JNDI 는, 다음의 패키지로 구성됩니다.
  • javax.naming
      네임 서비스에 액세스하기 위한 , 클래스와 인터페이스가 포함된다

  • javax.naming.directory
      코어 javax.naming 패키지를 확장해, 네임 서비스 뿐만이 아니라 디렉토리에 액세스 할 수 있는 기능을 제공한다

  • javax.naming.event
      네임 및 디렉토리 서비스내에서 이벤트 통지를 행하기 위한, 클래스와 인터페이스가 포함된다

  • javax.naming.ldap
      LDAPv3 의 확장 조작과 컨트롤을 지원하기 위한 , 클래스와 인터페이스가 포함된다

  • javax.naming.spi
      다양한 네임 및 디렉토리 서비스 프로바이더를 JNDI 환경에 동적으로 플러그 인하기 위한 , 클래스와 인터페이스가 포함된다
