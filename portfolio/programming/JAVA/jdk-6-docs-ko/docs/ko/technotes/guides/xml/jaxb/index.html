Java Architecture for XML Binding (JAXB)

문서의 목차

JavaTM Architecture for XML Binding (JAXB)는, XML 문서와 Java 객체간의 매핑을 자동화하는 API 와 툴을 제공하고 있습니다.   

JAXB 시스템를 사용하면(자), 개발자는 다음의 조작을 실행할 수 있습니다.

  • XML 컨텐츠를 Java 표현에비정렬화합니다
  • Java 표현에액세스해갱신합니다
  • XML 컨텐츠의 Java 표현을 XML 컨텐츠에정렬화합니다

JAXB 는, XML 와 Java 코드간의 매핑을 행하기 위한 효율적으로 표준적인 방법을 Java 개발자에게 제공하고 있습니다. JAXB 를 사용하는 Java 개발자는, 스스로 코드를 기술하는 양을 줄일 수가 있어 XML 에 정통할 필요가 없기 때문에, 생산성이 향상합니다. JAXB 를 사용하면(자), 개발자는 XML 및 Web 서비스 기술을 사용해 어플리케이션의 기능을 확장할 수 있습니다.
