java.lang* 패키지와 java.util. * 패키지

 문서의 목차
  • java.lang - Java 프로그램 언어의 설계의 기초가 되는 클래스를 제공합니다.
  • java.lang.annotation - Java 프로그램 언어의 주석 기능의 라이브러리 지원를 제공합니다.
  • java.lang.instrument - Java 프로그램 언어 에이전트에 의해 JVM 로 실행중의 프로그램을 계측 하는 서비스를 제공합니다.
  • java.lang.management - Java 가상 머신 및 Java 가상 머신을 실행하고 있는 operating system의 감시와 관리를 행하기 위한, 관리 인터페이스를 제공합니다.
  • java.lang.ref - 참조 객체 클래스를 제공해, 한정된 레벨에서의 가비지 컬렉터와의 대화를 가능하게 합니다.
  • java.lang.reflect - 클래스와 객체에 관한 리플렉트 정보를 취득하기 위한 , 클래스와 인터페이스를 제공합니다.
  • java.util - Collections Framework, 구버젼의 컬렉션 클래스, 이벤트 모델, 일자와 시간 기능, 국제화, 그 외의 유틸리티 클래스 (캐릭터 라인 토크나이자, random number generation 함수, 비트 배열)가 포함됩니다.
  • java.util.concurrent - 병행 프로그래밍으로 일반적으로 유용한 유틸리티 클래스.
  • java.util.concurrent.atomic - 단일 변수에 의한 락 프리로 thread 프리인 프로그래밍을 지원하는 클래스의 작은 툴 킷.
  • java.util.concurrent.locks - 편입의 동기와 모니터와는 다른 조건을 락 대기하기 위한 시스템를 제공하는 인터페이스와 클래스.
  • java.util.jar - JAR (Java ARchive) 파일 형식을 읽고 쓰기하기 위한 클래스를 제공합니다. JAR 파일은 기본적으로, 옵션의 Manifest 파일이 포함된 표준 ZIP 파일 형식입니다.
  • java.util.logging - Java 2 플랫폼의 핵심적인 로그 기능의 클래스와 인터페이스를 제공합니다.
  • java.util.regex - 문자 순서를 정규 표현으로 지정된 패턴과 일치시키기 위한 클래스.
  • java.util.prefs - 이 패키지를 사용하면, 사용자 및 시스템의 설정과 구성 데이터를 어플리케이션으로 포함 및 취득할 수 있습니다.
  • java.util.spi - java.util 패키지의 클래스의 서비스 프로바이더 클래스.
  • java.util.zip - 표준 ZIP 및 GZIP 파일 형식을 읽고 쓰기하기 위한 클래스를 제공합니다.
