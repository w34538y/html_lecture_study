Java Platform Standard Edition (Java SE) 6 에서의 XML

 문서의 목차

XML (Extensible Markup Language)는, 공통의 정보 서식을 작성해 World Wide Web, 인트라넷등으로 서식과 데이터를 공유하기 위한 유연한 방법입니다. XML 는 일관한 방법으로 정보를 공유할 필요가 있는 개인이나 기업에서 사용할 수가 있습니다. XML 는 World Wide Web Consortium (W3C )에 의해 정식으로 권고된 기술 언어이며, Web 페이지로 현재 넓게 사용되고 있는 Hypertext Markup Language (HTML)와 닮아 있습니다. XML 에도 HTML 에도, 페이지 (파일)의 내용을 기술하는 마크 업 기호가 있습니다. 다만 HTML 에서는, Web 페이지의 내용 (주로 텍스트와 그래픽 이미지)을, 표시 방법과 상호작용에 대한 보고 기술합니다. 예를 들어, <P> 로 새로운 단락이 시작됩니다. XML 에서는, 어떠한 데이터가 기술되는가 하는 관점으로부터 내용을 기술합니다. 예를 들어, <PHONENUM> 에 계속되는 데이터로 전화번호를 나타낼 수가 있습니다. 즉, XML 파일은 순수하게 데이터로서 프로그램으로 처리하거나 다른 컴퓨터에 같은 데이터와 함께 보존하거나 할 수가 있습니다. 또, HTML 파일과 같이 표시할 수도 있습니다. 예를 들어, 수신측의 컴퓨터의 어플리케이션이 요구하는 전화번호의 처리 방법에 근거한, 전화번호의 보존, 표시, 다이얼등이 가능합니다.
