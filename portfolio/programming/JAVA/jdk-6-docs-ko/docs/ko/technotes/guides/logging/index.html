JavaTM 로깅테크노로지

문서의 목차

Java TM 로깅 API 는 java.util.logging 패키지의 형태로 도입되어 최종 사용자, 시스템 관리 책임자, 필드 서비스 엔지니어, 및 소프트웨어의 개발 팀이 분석하기 위한 로그 리포트를 작성해, 고객 사이트에서의 소프트웨어의 서비스나 보수를 간단하게 할 수 있도록(듯이) 하는 것입니다. Logging API 에서는, 어플리케이션 또는 플랫폼에서 발생한 시큐리티의 실패, 구성 에러, 퍼포먼스의 장해, 버그등의 정보를 수중에 넣습니다. 주요 패키지에서는, plaintext 또는 XML 형식의 로그 레코드를 메모리, 출력 스트림, 콘솔, 파일, 및 소켓에 전달합니다. Logging API 에는, 호스트 operating system상의 기존의 로그 서비스라고 대화하는 기능이 있습니다.
