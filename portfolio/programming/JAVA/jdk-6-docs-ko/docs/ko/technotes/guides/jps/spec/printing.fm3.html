인쇄 작업의 취득

인쇄 작업이란, 송신된 인쇄 요구입니다. 인쇄 작업은, 1 개(살) 또는 복수의 인쇄 데이터 및 일련의 처리 명령으로 구성됩니다. Java 인쇄 서비스는, DocPrintJob 객체를 사용해 인쇄 작업을 표현합니다. 인쇄 데이터의 송신지 (스트림 또는 프린터)에 관계없이, 인쇄 작업을 작성할 때는, 서비스상에서 createPrintJob 를 호출합니다. 

	DocPrintJob pj = pservices[0]. createPrintJob();
어플리케이션은, 인쇄 작업을 인쇄 서비스로부터 가져옵니다. 인쇄 서비스가 작성한 인쇄 작업이 아니면, 그 서비스가 받아들이는 데이터를 처리할 수 없기 때문입니다.

DocPrintJob 인터페이스에는, 인쇄 메소드가 준비되어 있습니다. 이 인쇄 메소드에는, PrintRequestAttributeSet 파라미터, 및 인쇄 데이터와 드큐먼트후레이바를 캡슐화한 Doc 를 지정합니다. 

	pj.print(doc, aset);
다음의 마디에서는, Doc 의 작성 방법에 대해 설명합니다.
