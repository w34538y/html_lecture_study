JavaTM Plug-in

 문서의 목차
JavaTM Plug-in 를 사용하면(자), Java SE Platform 6 스펙에 따라 기술된 애플릿을 Mozilla 및 Internet Explorer Web 브라우저상에서 실행할 수 있습니다. Java Plug-in 는, 6 JRE/JDK 에 부속되어 있습니다.
Java Plug-in 개발자 가이드
「Java Plug-in 개발자 가이드 (일본어판)」 에는, 다음의 내용이 포함됩니다.
  • Java Plug-in 의 기본
  • 배치 schema
  • 시큐리티
  • 디버그의 지원
  • 상급자전용의 토픽
공통 DOM API
애플릿 개발자는, Java Plug-in 를 사용해 실행하는 애플릿을 기술하는 경우,공통 DOM (Document Object Model) API 를 사용할 수 있습니다. 공통 DOM API 를 사용하면(자), 애플릿으로부터 DOM 를 더듬는 것이 가능하게 됩니다. 공통 DOM API 는, Java Plug-in 의 1.4, 5.0, 또는 6 버젼으로 실행되는 애플릿에서만 지원되고 있습니다.
