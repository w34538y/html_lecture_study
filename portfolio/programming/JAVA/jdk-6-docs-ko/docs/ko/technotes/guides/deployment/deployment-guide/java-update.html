Java Update

JRE 를 인스톨 하면(자), Java Update 기능이 자동적으로 인스톨 됩니다. Java Update 를 사용하면(자), 최신의 JRE 릴리스로, 사용의 컴퓨터를 자동적으로 최신 상태에 유지할 수가 있습니다. 중요한 JRE 갱신을 찾을 필요는 이제 없습니다. Java Update 에 의해, JRE 의 최신 버젼이 직접사용의 컴퓨터에 전달됩니다. Java Update 에서는, 언제 온라인일까를 판단해, 인터넷 접속을 사용해, 이용 가능한 갱신을 Java Update Web 사이트에서 검색합니다. 갱신이 이용 가능한 때는, 태스크바의 상태 영역에 아이콘이 표시됩니다.

Java Update 로 언제 어떻게 사용의 컴퓨터를 갱신하는지를 지정할 수 있습니다. 예를 들어, 지정한 스케줄에 맞추어 자동적으로 다운로드해 인스톨 하도록(듯이) 설정할 수 있습니다. 또, 갱신의 다운로드와 인스톨의 전에 통지를 받도록(듯이) 하거나 다운로드와 인스톨의 사이에만 통지를 받도록(듯이) 하는 일도 가능합니다. 게다가 자동 갱신의 옵션을 완전하게 무효로 해, 수동으로 갱신하도록(듯이)도 할 수 있습니다. 옵션은 사용자를 자유롭게 선택할 수 있어 언제라도 재설정할 수 있습니다.

자세한 것은,「Java 컨트롤 패널」의「[업데이트] 패널」을 참조해 주세요.
