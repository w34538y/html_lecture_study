문서의 목차

BMP 및 WBMP 형식의 Image I/O 지원

이 변경에 관련하는 버그 리포트: 4339462 및 4339384

이 변경에 의해, BMP 및 WBMP 형식의 read 플러그 인 및 기입해 플러그 인이 Image I/O 시스템에 추가됩니다. J2SE 5.0 에서는,javax.imageio 의 모든 구현에 의해, JPEG, PNG, BMP, WBMP 이미지의 read 및 기입해, 또, GIF 이미지의 읽을 지원가 제공됩니다.

상세한 것에 대하여는,javax.imageio.plugins.bmp 패키지 및 javax.imageio.plugins.bmp.BMPImageWriteParam 클래스의 API 문서를 참조해 주세요. 한층 더 자세한 정보에 대해서는,BMP 및 WBMP 형식의 메타데이타에 관한 설명을 참조해 주세요.

퍼포먼스의 향상

이 변경에 관련하는 버그 리포트:4827358 ,4867874 , 및 4868479

이러한 변경에 의해, JPEG 이미지의 read 및 기입해에 있어서의 Image I/O system 의 퍼포먼스와 메모리 사용량이 개선됩니다. 이 변경은 Image I/O API 에 영향을 주지 않습니다.
