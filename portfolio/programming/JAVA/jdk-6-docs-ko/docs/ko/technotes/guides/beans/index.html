JavaBeansTM Component API

 문서의 목차
JavaBeans 컴퍼넌트 API 는, Java 플랫폼의 「Write Once, Run AnywhereTM」기능을 확장해, 재사용 가능한 컴퍼넌트 개발을 가능하게 합니다. 실제의 곳, JavaBeans 는, 상호 운용성을 크고 한 걸음 전진시킵니다. 기술한 코드는, 모든 OS 상에서, 그리고 모든 어플리케이션 환경내에서 가동합니다. Bean 의 개발자는, 독자적인 플랫폼을 사용하는 고객을 잃는 일 없이, 새롭게 계속 대두되고 있는 네트워크 소프트웨어 마켓에서의 장래를 확실한 것으로 할 수 있습니다. 이것은, JavaBeans 가 ActiveX, OpenDoc, 및 LiveConnect 와의 상호 운용이 가능하기 때문에입니다. JavaBeans 의 장기 지속성에 의해 프로퍼티치를 나타내는 텍스트를 사용해 Bean 를 읽고 쓰기하는 것이 가능합니다.
