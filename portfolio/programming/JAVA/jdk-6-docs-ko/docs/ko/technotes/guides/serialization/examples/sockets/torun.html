소켓과 직렬화의 사용

예제의 실행

다음의 순서로, 2 개의 예제를 컴파일 해 실행합니다.

  1. 자신의 베이스 디렉토리아래의 objectserializationexamples/UsingSockets 디렉토리에서, JDKTM1.1 이후를 사용해 Server.java 와 Client.java 를 컴파일 합니다. 
      javac Server.java
  javac Client.java
  2. Server 를 실행합니다. 
      java Server
  3. Server 를 중단합니다. UNIX 시스템의 경우, Ctrl+Z 키를 사용합니다.
  4. Server 를 백그라운드에서 실행시킵니다. UNIX 시스템의 경우, 다음의 커멘드를 사용합니다. 
      bg
  5. Client 를 실행합니다. 
      java Client

소켓과 직렬화를 사용해, 클라이언트로부터 서버에 현재의 일자가 보내져 다음에, 서버가 그 일자를 출력합니다.
