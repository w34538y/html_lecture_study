AWT 패키지

 문서의 목차
  • java.awt - 사용자 인터페이스의 작성 및 그래픽스와 이미지의 페인트를 실시하는 모든 클래스가 포함된다.
  • java.awt.event - AWT 컴퍼넌트에 트리거되는 각종의 이벤트 타입을 처리하는 인터페이스와 클래스를 제공한다.

다음의 패키지는 AWT 패키지에 포함되어 있어드래그＆드롭을 지원하기 위해서 사용됩니다.

  • java.awt.datatransfer - 어플리케이션간이나 어플리케이션내의 데이터 전송을 실시하는 인터페이스와 클래스를 제공한다.
  • java.awt.dnd - 드래그＆드롭은, 많은 그래피컬 사용자 인터페이스 (GUI) 시스템에 보여지는 직감적인 조작 방법으로, GUI 내의 표시 요소에 논리적으로 관련지을 수 있었던 2 개의 원가요소간에 정보를 전송하기 위한 기구를 제공한다.

다음의 패키지는 AWT 패키지에 포함되어 있습니다만, 실질적으로는 Java 2D 의 일부입니다.

  • java.awt.color
  • java.awt.font
  • java.awt.geom
  • java.awt.image
  • java.awt.image.renderable
  • java.awt.print

다음의 패키지는 AWT 패키지에 포함되어 있습니다만, 실질적으로는 Input Method Framework 의 일부입니다.

  • java.awt.im
  • java.awt.im.spi
