참조

All packages that begin with org.omg are part of the Java IDL API. 그 중에서 이하의 패키지가 중요합니다.

  • 패키지 org.omg.CORBA - OMG CORBA API 의 Java[TM] 프로그램 언어에의 매핑을 제공한다
  • 패키지 org.omg.CosNaming - Java IDL 의 네임 서비스를 제공한다
  • 패키지 org.omg.PortableServer - 어플리케이션의 서버측을 복수 벤더의 ORB 간에 이식 가능하게 하는 클래스와 인터페이스를 제공한다
  • 패키지 org.omg.PortableInterceptor - ORB 훅을 등록하는 기구를 제공해, ORB 서비스가 그 훅에 의해 ORB 의 실행의 일반적으로의 흐름을 차단할 수 있도록(듯이) 한다
  • 패키지 org.omg.DynamicAny - DynAny 객체를 통해서 Any 치를 동적으로 해석 (처리)해 구축할 수 있도록(듯이) 하는 클래스와 인터페이스를 제공한다
  • org.omg.CORBA.ORB - CORBA Object Request Broker 기능에 API 를 제공한다
