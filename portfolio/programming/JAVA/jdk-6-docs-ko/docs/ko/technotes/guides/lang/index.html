java.lang* 패키지와 java.util. * 패키지

 문서의 목차

java.lang 및 java.util 패키지에는, 거의 모든 어플리케이션이 사용하는 기본 기능이 준비되어 있습니다. java.lang 및 java.util 로 시작되는 이름이 붙은 다른 패키지는, 특정 기능용의 추가 API 를 제공합니다.
