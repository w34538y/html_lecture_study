슈퍼 클래스가 직렬화 가능하지 않은 경우의 직렬화

예제의 실행

다음의 순서로, 2 개의 예제를 컴파일 해 실행합니다.

  1. 자신의 베이스 디렉토리아래의 objectserializationexamples/NonSerializableSuperClass 디렉토리에서 JDKTM 1.1 이후를 사용해, 다음의 파일을 컴파일 합니다. 
      javac NonSerialSuperExample.java
  javac Book.java
  javac ReadingMaterial.java
  2. 컴파일 한 파일을 실행합니다. 
      java NonSerialSuperExample

직렬화전과 직렬화 후에 Book 객체가 출력됩니다.
