javac - Java™ 프로그램 언어 컴파일러

 문서의 목차
Java 프로그램 언어 컴파일러인 javac 는, Java 프로그램 언어로 기술된 원시 파일을 읽어내, 바이트 코드의 클래스 파일에 컴파일 합니다.  필요에 따라서, 플러그 인 가능한 주석 처리 API 를 사용해, 원시 파일과 클래스 파일로부터 검출된 주석을 처리할 수도 있습니다.  이 컴파일러는 커멘드행 툴입니다만, Java 컴파일러 API 를 사용해 호출할 수도 있습니다.
이 컴파일러는, JLS (Java 언어 스펙)로 정의된 원시 코드를 받아들여 JVMS (JAVA 가상 머신 스펙)로 정의된 클래스 파일을 생성합니다.

API 의 스펙

  • javax.annotation.processing - 주석 처리.
  • javax.lang.model - 주석 처리에 사용되는 언어 모델과 컴파일러 트리 API
    • javax.lang.model.element - 언어 요소.
    • javax.lang.model.type - 타입.
    • javax.lang.model.element - 언어 모델 유틸리티.
  • javax.tools - Java 컴파일러 API.
  • com.sun.source. * - 컴파일러 트리 API.

상세 정보
