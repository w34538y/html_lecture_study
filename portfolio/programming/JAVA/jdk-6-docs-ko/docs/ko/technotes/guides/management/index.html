Java 플랫폼의
감시와 관리

 문서의 목차

Java Platform Standard Edition (Java SE) 6 은, Java 플랫폼의 감시 및 관리를 포괄적으로 지원합니다.  

API 스펙

Java SE 6 에는 감시와 관리를 실시하기 위해서(때문에) 다음의 API 가 포함되어 있습니다.

  • java.lang.management - Java 가상 머신 및 그 기반이 되는 operating system의 감시와 관리를 유효하게 합니다.  이 API 에 의해, 어플리케이션에 의한 자기 감시가 가능하게 되어, JMX 에 준거한 툴로 가상 머신의 감시와 관리를 로컬 및 원격에 실시할 수가 있습니다.

  • com.sun.management - Sun Microsystems 플랫폼용의,java.lang.management API 및 다른 컴퍼넌트용의 관리 인터페이스의 확장 기능.

  • java.util.logging.LoggingMXBean - 로그 정보의 취득과 설정을 할 수 있습니다.

  • Java Management Extensions (JMX) API - Java 의 어플리케이션과 네트워크의 관리 및 감시의 아키텍쳐(architecture), 디자인 패턴, 인터페이스, 및 서비스를 정의합니다.  자세한 것은,「JMX 문서」를 참조해 주세요.

  • Attach API - 관리 에이전트를 가상 머신에 동적으로 로드할 수 있는, Sun Microsystems 플랫폼용의 확장 기능.

  • JConsole API - JConsole 에 액세스 하는 프로그램 인터페이스 (JConsole 플러그 인의 추가등)를 제공하는, Sun Microsystems 의 플랫폼 확장 기능. JTop 는 JConsole 플러그 인의 1 예로,JDK_HOME/demo/management/JTop 디렉토리에 있습니다.
