슈퍼 클래스가 외부화 가능한 경우의 외부화

예제의 실행

이 예제를 컴파일 해 실행하려면 , 다음의 순서에 따릅니다.

  1. 자신의 베이스 디렉토리아래의 objectserializationexamples/ExternalizableSuperclass, 디렉토리에서, JDKTM 1.1 이후를 사용해 Savesuper.java 를 컴파일 합니다. 
      javac Savesuper.java
  2. 컴파일 한 파일을 실행합니다. java Savesuper

원의 book 객체 (직렬화전의 객체)가 출력되어 다음에 새로운 book 객체 (직렬화 후의 객체)가 출력됩니다.
