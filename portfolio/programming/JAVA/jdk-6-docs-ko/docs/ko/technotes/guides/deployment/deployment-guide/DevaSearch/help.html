복수의 키워드를 입력하는 경우는 사이에 스페이스를 삽입합니다. 다음에 나타내는 것을 제외해, 검색 엔진은 리스트의 임의의 키워드를 포함한 페이지를 검색합니다 (boolean 의 OR 를 사용). 키워드를 더하는 만큼, 검색 결과는 많아집니다.

검색 키워드에 대문자 소문자의 구별은 없습니다. 키워드의 입력시에는, 대문자와 소문자를 어떻게 조합해도 괜찮습니다.

키워드에는 다음의 문자를 포함할 수가 있습니다. A-Z a-z 0-9 _ - . ' , $ %.

검색 대상으로 하는 단어의 머리에는 플러스 기호 (+)를 붙입니다. 예를 들어,+printer +cartridge +maintenance 라고 입력했을 경우, 검색 엔진은 이것들 3 개의 단어가 모두 존재하는 페이지를 찾아냅니다. 이것들 3 개의 단어는 반드시 인접하고 있는 것은 아닙니다만, 같은 페이지에 존재하는 것입니다.

검색 대상으로부터 떼는 단어의 머리에는 마이너스 기호 (-)를 붙입니다. 예를 들어,+printer +cartridge -maintenance 라고 입력했을 경우, 검색 엔진은 「printer」와「cartrige」를 포함해, 한편 「maintenance」를 포함하지 않는 페이지를 찾아냅니다.

공통의 단어 (a, an, the 등)는 검색 인덱스로부터 제외됩니다.
