JavaTM API for XML Processing (JAXP)

Java API for XML Processing (JAXP)를 사용하면(자), 특정의 XML 프로세서 구현에 의존하지 않는 API 를 사용해, 어플리케이션으로 XML 문서의 해석, 변환, 검증, 및 문의를 실시할 수가 있습니다. JAXP 에서는, 플러그 인 가능 층이 제공되고 있습니다. 이것을 사용하면(자), 벤더는 어플리케이션 코드내에 의존관계(dependencies)의 문제를 발생시키는 일 없이, 독자적인 구현을 제공할 수 있습니다. 이 소프트웨어를 사용하면(자), 어플리케이션이나 툴의 개발자는, 전자 상거래, 어플리케이션 통합, 및 Web 퍼블리싱용으로 완전한 기능을 갖춘 XML 대응의 Java 어플리케이션을 구축할 수 있습니다.

Java Platform, Standard Edition version 6.0 에는 JAXP 1.4 가 포함되어 있습니다. JAXP 1.4 는, Streaming API for XML (StAX)를 지원하는 JAXP 1.3 의 보수의 릴리스입니다. 현재의 개발 상황에 대해서는,Java.net 의 JAXP 프로젝트를 참조해 주세요.

피드백

JAXP 의 메일링리스트를 사용해 피드백을 송신할 수 있습니다. 일반적인 질문에 관한 메세지는,users@jaxp.dev.java.net 에 투고해 주세요. 문제나 버그에 관한 메세지는,issues@jaxp.dev.java.net 에 투고해 주세요. 메세지를 송수신 하려면 , 메일링리스트를 구독할 필요가 있는 경우가 있습니다.
