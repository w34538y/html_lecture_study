제 2 장

속성




인쇄 데이터를 클라이언트로부터 프린터에 송신하면(자), 사용하는 매체 (용지)와 인쇄 부수 등, 인쇄 방법에 관한 정보도 송신됩니다. 클라이언트상에서는, JavaTM 인쇄 서비스 API 의 속성 정의를 사용해, 이러한 처리 명령을 지정합니다.

Java 인쇄 서비스 API 에는, javax.print.attribute 및 javax.print.attribute.standard 라고 하는, 속성을 정의하는 2 개의 패키지가 짜넣어지고 있습니다. javax.print.attribute 패키지는, 속성의 종류와 그것들을 수집해 속성 세트에 정리할 방법이 기술된 인터페이스와 클래스로 구성됩니다. javax.print.attribute.standard 패키지에는, 이 API 내의 표준 속성이 모두 열거되고 있습니다. JPS API 의 속성 정의의 대부분은, IETF 의 IPP (Internet Printing Protocol)에 의해 정의된 표준 및 확장 속성의 구현입니다.

이 장에서는, JPS API 의 속성의 정의를 상세하게 설명합니다. 「속성의 카테고리와 값」으로는,JPS 로 사용하는 속성에 대해 설명합니다. 「속성의 기능」에서는, 속성의 종류에 대해 설명합니다. 「속성 세트」에서는, 속성을 수집해 속성 세트에 정리하는 방법에 대해 설명합니다. 「표준 속성」에서는,javax.attribute.standard 패키지에 정의되고 있는 사용 빈도의 높은 속성에 대해 설명합니다.
