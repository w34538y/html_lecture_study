직렬화와 직렬화 가능 필드 API 의 사용

개설

이 예제는, 직렬화 가능 필드 API 를 사용한 직렬화의 방법을 나타냅니다. 「클래스가, 클래스내의 기존의 필드 이외의 필드를 직렬화 가능하게 할 수 있다」일의 실례입니다. 이 예제에서는 버젼 관리가 지원되고 있어 단지 writeObject 메소드를 고쳐 써 데이터 형식을 커스터마이즈 하는 것으로는 다릅니다 (「커스텀 데이터 형식과 직렬화의 사용」을 참조).

이 예제에서는, 직렬화 가능 필드 API 를 사용해, 사각형의 내부 표현을 x1, y1, x2, y2 의 형식 (OriginalClass.java 를 참조)으로부터 Point(x1, y1), Point(x2, y2) (EvolvedClass.java 를 참조)로 변경합니다만, 외부 표현은 x1, y1, x2, y2 인 채입니다. 이것에 의해, 오리지날의 표현과 전개된 표현동안에 쌍방향의 호환성이 확보됩니다.

전개를 취급하는 다른 예제와 같게, 전개 후의 클래스 (ARectangle)에는, 오리지날 클래스 (ARectangle)에 커멘드 serialver 를 사용해 취득한 serialVersionUID 필드를 정의할 필요가 있습니다.
