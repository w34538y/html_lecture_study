1.3 플랫폼

이 API 는, Java Platform Standard Edition 의 「Merlin」릴리스 (J2SE 버젼 1.4)의 표준 패키지에 포함되어 있습니다. Sun 는, Java SE 플랫폼 폼의 구현을, Solaris 오퍼레이팅(operating) 환경, Linux, 및 Microsoft Windows 의 각 플랫폼에 대해 제공할 예정입니다. 그 외의 플랫폼에서 특정의 Java SE 릴리스가 이용 가능한가 어떤가는, 플랫폼의 벤더에 의해 결정됩니다.
