serialver - 시리얼 버젼 커멘드

serialver 커멘드는 serialVersionUID 를 돌려줍니다.

형식

serialver [ options  ]

설명

serialver 는, 1 개(살) 이상의 클래스의 serialVersionUID 를, 전개하고 있는 클래스에 카피하는데 적합한 형식에서 돌려줍니다. 인수를 지정하지 않고 호출하면(자), 사용 방법이 표시됩니다.

옵션
-show

간단한 사용자 인터페이스를 표시합니다. 완전 지정의 클래스명을 입력해, Enter 키나 [Show] 버튼을 눌러, 시리얼 버젼 UID 를 표시합니다.

관련 항목

java.io.ObjectStreamClass
