JavaTM 2 플랫폼의 java.io.File 의 기능

 JDKTM 목차
File 클래스는, 파일 및 디렉토리 경로명의 추상 표현입니다. 새로운 File 클래스의 스펙에서는, Windows 등의 UNIX 이외의 플랫폼이 보다 강력하게 지원되게 되었습니다. Windows 에서는, 드라이브 문자, UNC 공유명, 및 복수의 단락 문자가 포함되는 경로명 구문이 풍부하게 사용되고 있습니다. JavaTM 2 플랫폼 이전의 File 클래스의 스펙에서는, 간단한 캐릭터 라인 조작을 실시하도록(듯이) 작성되고 있었기 때문에, 이러한 문자에는 대응하고 있었었습니다.

다음의 기능이, v1. 2 의 Java 2 플랫폼의 File 클래스에서 도입되었습니다.

  • 임시 파일의 작성 기능, 및 간단한 delete-on-exit 기능 (새로운 메소드 createNewFile ,createTempFile , 및 deleteOnExit )
  • 추상 경로명으로부터 URL 에의 변환 (새로운 메소드 toURL )
  • 파일 시스템 루트의 일람에의 액세스 (새로운 메소드 listRoots )
  • 비교 기능 (새로운 compareTo 메소드)
  • 파일의 추가 속성에의 액세스 (새로운 메소드 isHidden ,setLastModified , 및 setReadOnly )
  • 캐릭터 라인은 아니고, File 객체의 송수신을 실시하는 유니폼 메소드 세트 (새로운 메소드 getParentFile ,getAbsoluteFile ,getCanonicalFile , 및 listFiles , 새로운 인터페이스 FileFilter )
