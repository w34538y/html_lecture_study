RMI-USERS 메일링리스트의 구독신청과 구독 중지


구독신청:
  • 전자 메일의 송부처:listserv@java.sun.com
  • 메세지의 내용:subscribe RMI-USERS

    구독신청 완료 후, RMI-USERS 의 매일의 메일 모든 것을 몇통의 메일에 정리하고 받는 방법을 선택할 수도 있습니다.

    • 전자 메일의 송부처:listserv@java.sun.com
    • 메세지의 내용:set RMI-USERS digest

    나중에 모든 메일을 개별적으로 수신할 수도 있습니다.

    • 전자 메일의 송부처:listserv@java.sun.com
    • 메세지의 내용:set RMI-USERS nodigest
구독 중지:
  • 전자 메일의 송부처:listserv@java.sun.com
  • 메세지의 내용:signoff RMI-USERS

RMI-USERS 의 전자 메일 어카이브(archive)를 조사하려면 ,여기를 클릭합니다.
