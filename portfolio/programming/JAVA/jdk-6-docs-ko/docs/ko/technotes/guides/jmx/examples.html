JMX (Java Management Extensions)의 예

 문서의 목차

다음에 일람 하는 파일은, JMX 테크놀러지의 주요한 기능의 몇개인가를 나타내는 코드예입니다. 이러한 예를 사용해, 보다 복잡한 MBeans 및 완전한 기능을 갖춘 JMX 에이전트를, 관리 솔루션에 맞추어 개발할 수가 있습니다.

각각의 예는, Java 원시 파일과 README 파일로부터 구성되어 있습니다. README 파일은, 예로 다루어지는 토픽과 클래스의 컴파일과 실행의 순서를 설명합니다.

또 JMX 테크놀러지의 모든 예를 거둔 Zip 파일도 다운로드할 수 있습니다. 예의 실행 방법의 자세한 것은,「Java Management Extensions (JMX) 테크놀러지의 튜토리얼」을 참조해 주세요.

주 - 여기서 가리키는 예의 그 밖에도, Java JDK (Development Kit) 6 을 인스톨 하면, 다음의 디렉토리에서 JMX API 의 실제의 구현을 나타내는 산프리아프리케이션을 볼 수가 있습니다.

JDK_HOME/sample/jmx/jmx-scandir

상기의 패스 가운데,JDK_HOME 가, JDK 소프트웨어가 인스톨 되는 디렉토리입니다. jmx-scandir 의 예는, 실제의 시나리오에 있어서의 JMX API 의 고도의 개념을 나타내는 예입니다.

여기에서는 ZIP 파일을 샘플로 해 JMX 테크놀러지의 예를 소개합니다.
