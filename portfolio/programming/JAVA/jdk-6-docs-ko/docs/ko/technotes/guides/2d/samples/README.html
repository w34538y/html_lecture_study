Java 2DTM 개발자 안내 - 이미징의 샘플 프로그램

이 샘플 프로그램에서는, JavaTM 2 SDK 버젼 1.2 로 이용할 수 있는 새로운 Java 2DTM 이미징 기능에 대해 소개합니다. 이러한 프로그램을 실행하려면 ,Java 2 SDKTM 1.2 또는보다 새로운 버젼의 SDK를 다운로드할 필요가 있습니다.

샘플 프로그램의 다운로드

이미징의 샘플 프로그램과 필요한 이미지를 모두 포함한 zip 파일을 다운로드하려면 , 다음의 링크를 클릭합니다.

  • Java 2D 이미징 프로그램 및 이미지의 다운로드 [zip]

샘플 프로그램의 컴파일

샘플의 . java 파일에 대해서 javac 를 실행하는 것으로, 샘플을 컴파일 할 수 있습니다. 예를 들어,Blur 샘플을 컴파일 하려면 , 다음과 같이 입력합니다.

javac Blur.java

(javac 가 실행 가능 패스에 있어, 한층 더 Java 2 SDK 1.2 FCS 이후를 사용하고 있을 필요가 있습니다.

샘플 프로그램의 실행

이러한 샘플 프로그램은, 스탠드얼론의 Java 어플리케이션 또는 애플릿으로서 설계되고 있습니다. 샘플을 어플리케이션으로서 실행하려면 , 클래스명에 대해서 java 커멘드를 실행합니다. 예를 들어,Blur 어플리케이션을 실행하려면 , 다음과 같이 입력합니다.

javac Blur.java

(javac 가 실행 가능 패스에 있어, 한층 더 Java 2 SDK 1.2 FCS 이후를 사용하고 있을 필요가 있습니다.

특정의 샘플의 html 파일에 대해서 appletviewer 를 실행하는 것에 의해, 샘플을 애플릿으로서 실행할 수도 있습니다. 예를 들어,Blur 애플릿을 실행하는 경우,html 파일과 같은 디렉토리로 이동해, 다음과 같이 입력합니다.

appletviewer Blur.html

(appletviewer 가 실행 가능 패스에 있어, 한층 더 Java 2 SDK 1.2 FCS 이후를 사용하고 있을 필요가 있습니다.
