외부화와 전개

개설

이 예제는, Externalizable 인터페이스를 사용해 지속성을 가지는 형식을 전개하는 방법을 나타냅니다. Externalizable 인터페이스를 사용하는 클래스는, 스스로의 상태의 보존, 슈퍼타입 상태의 보존, 및 버젼 관리 (데이터의 스킵을 포함한다)의 책임을 담당합니다.

이 예제에서는, 특히 버젼 관리를 취급합니다. 슈퍼타입 (외부화가 가능한 경우와 불가능한 경우가 있다) 상태의 보존의 자세한 것은, 다음의 예제를 참조해 주세요.

  • 슈퍼 클래스가 직렬화 가능하지 않은 경우의 직렬화
  • 슈퍼 클래스가 외부화 가능한 경우의 외부화
  • 슈퍼 클래스가 외부화 가능하지 않은 경우의 외부화
