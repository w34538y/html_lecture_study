JavaTM Sound Technology

 문서의 목차
Java 플랫폼에는, 오디오 및 MIDI (Musical Intstrument Digital Interface) 데이터의 수중에 넣어, 처리, 및 재생용의 강력한 API 가 포함되어 있습니다. 이 API 는, 이 플랫폼에서의 고품질인 오디오 믹싱 및 MIDI 합성을 보증하는 효과적인 사운드 엔진에 의해 지원되고 있습니다.

이 API 의 제공되는 레퍼런스 구현에서는, 다음의 기능이 지원되고 있습니다.

  • 오디오 파일 형식:AIFF, AU, 및 WAV
  • 음악 파일 형식 :MIDI Type 0, MIDI Type 1, 및 Rich Music Format (RMF)
  • 사운드 형식:단청의 또는 스테레오에서의 8 비트 및 16 비트의 오디오 데이터. 샘플링 레이트는 8 kHz 에서 48 kHz
  • 지원되고 있는 오디오 파일 형식에서의 선형, a-law, 및 mu-law encode 데이터
  • 소프트웨어내에서의 MIDI WaveTable 의 합성과 시켄싱, 및 하드웨어 MIDI 디바이스에의 액세스
  • 최대 64 채널의 디지털 오디오 및 합성된 MIDI 음악의 믹싱 및 렌더링이 가능한, 모든 것이 소프트웨어의 믹서

상기의 리스트는, API 자체는 아니고, 레퍼런스 구현에 대한 설명입니다. 이 API 에서는, 인스톨이 끝난 상태로 이용 가능한 자원에 대해 어플리케이션이 시스템에 문의하는 방법 등, 오디오 및 MIDI 시스템의 유연한 구성이 가능하게 되어 있습니다.
