JavaTM Web Start

 문서의 목차

Java Web Start 버젼 1.6. 0 은, Java TM Platform, Standard Edition 6 JDK 에 포함되어 있습니다.

개요

Java Web Start 는, Web 브라우저상에서 클릭하는 것만으로 풀 장비의 어플리케이션을 기동할 수 있는 기능을 제공하는, 어플리케이션 배치를 위한 테크놀러지입니다. 이 기능을 이용하면, 표계산 프로그램이나 인터넷 채팅 클라이언트등의 어플리케이션을, 복잡한 인스톨 순서를 밟지 않고 , 간단하게 다운로드해 기동할 수 있게 됩니다.

Java Web Start 에는, Java 플랫폼의 시큐리티 기능이 짜넣어지고 있기 (위해)때문에, 어플리케이션의 데이터나 파일의 무결성이 손상될 것은 없습니다. 게다가 Java Web Start 테크놀러지를 사용하면(자), 최신의 Java SE 테크놀러지를 임의의 브라우저로 이용할 수 있습니다.

Java Web Start 에서는, Web 페이지상의 링크를 클릭하는 것만으로, 어플리케이션을 기동할 수 있습니다. 그 어플리케이션이 컴퓨터에 존재하지 않는 경우는, Java Web Start 에 의해, 필요한 모든 파일이 자동적으로 다운로드됩니다. 다운로드 후는, 파일이 컴퓨터상에 캐쉬되기 (위해)때문에, 데스크탑상의 아이콘이나 브라우저상의 링크를 사용해, 필요한 때에 언제라도 기동할 수 있게 됩니다. 또, 어느 방법으로 어플리케이션을 기동해도, 항상 최신 버젼의 어플리케이션이 기동됩니다.
