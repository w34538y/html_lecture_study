상세 정보

  • 네트워크의 개요 (일본어판) (PDF (영문판) 에서도 입수 가능)
    java.net 의 네트워크 기능의 개요. 주소 지정, TCP 접속의 확립, UDP 를 사용한 데이터 그램 패킷의 송수신, 네트워크 자원의 검색과 식별, 시큐리티 등

  • HTTP 클라이언트 API. 다음의 기사에서는 HTTP 클라이언트 API 의 기능에 대해 설명합니다 (개요에 대해서는 java.net.HttpURLConnection 를 참조해 주세요).

    • URLConnection 캐쉬 API
      새로운 캐쉬 관리 API 의 개요를, 코드 샘플을 들면서 상세하게 설명합니다.

    • Cookie 관리
      J2SE 5.0 에 대해 HTTP 상태 관리 기구로서 사용되는 경우가 많은 Cookie 처리 방법의 개요를 설명합니다.

    • HTTP 인증
      HTTP 인증의 사용 방법, 지원되는 방식 및 프로토콜, 동작에 영향을 미치는 시스템 프로퍼티에 대해 개요를 설명합니다.

    • 지속 접속
      HTTP 지속 접속 (별명 킵얼라이브 기구) 및 그 제어 수단의 개요를 설명합니다.

  • Java 네트워크 및 프록시
    프록시 관리 기능의 Java 네트워크 API 에 대해, 코드 샘플을 사용하면서 상세한 검토를 실시합니다. 새롭게 도입된 ProxySelector 기구에 대해서도 검토합니다.

  • IPv6 의 지원 (일본어판)： (PDF (영문판) 에서도 입수 가능)
    IPv6 의 지원. 특수한 종류의 IPv6 주소나, 듀얼 스택 노드 등

  • 네트워크의 프로퍼티
    네트워크로 사용되는 프로퍼티에 대한 설명

  • Java 에 있어서의 정상형과 방폐형의 접속 해제
    네트워크 어플리케이션으로 확실히 TCP 접속을 정상 해제하는 방법에 대해

  • 확장 가능한 Socket 와 ServerSocket
    설계, 원리 및 그방법

  • 소켓의 옵션
    일반적인 BSD 스타일의 옵션의, Java base class에서의 지원

  • Socket 의 예외
