java.lang.instrument

문서의 목차

java.lang.instrument 패키지는, Java 프로그램 언어 어플리케이션을 계측 하는 툴 (예를 들어, Java 프로그램 언어 어플리케이션의 감시나 퍼포먼스 정보의 수집을 실시하는 툴)에 JavaTM 프로그램 언어 API 를 제공합니다. 툴은 java.lang.instrument 를 사용해, 클래스를 정의하는 클래스 파일을 수정합니다. 일반적으로, 클래스의 정의는, 계측을 실행하는 다른 바이트 코드를 메소드의 바이트 코드에 삽입해 실시합니다.

JDKTM 5.0 에는,java.lang.instrument 가 도입되었습니다.
