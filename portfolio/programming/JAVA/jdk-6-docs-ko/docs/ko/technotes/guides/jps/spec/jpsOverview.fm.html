제 1 장

처음에




JavaTM 인쇄 서비스 (JSP)는, 새로운 Java 인쇄 API 로, 사이즈가 제한되는 플랫폼을 포함해 모든 Java 플랫폼상에서 인쇄할 수 있도록(듯이) 설계되고 있습니다. 또, 현재의 JavaTM 2 인쇄 API 도 계속해 지원합니다. 이 통합된 Java 인쇄 API 에는, IETF 스펙 RFC 2911 의 IPP (Internet Printing Protocol)에 규정되고 있는 표준 속성에 근거한, 확장 가능한 인쇄 속성 세트가 짜넣어지고 있습니다. 클라이언트 및 서버의 어플리케이션은, 이러한 속성을 사용해, 속성으로 지정된 기능을 가지는 프린터를 검출 및 선택할 수가 있습니다. 새롭게 추가된 StreamPrintService 를 사용하면(자), 인쇄 데이터 서식을 어플리케이션으로 변환할 수가 있습니다. 또, 서비스 프로바이더 인터페이스를 사용하면, Sun 이외의 벤더가 독자적인 인쇄 서비스를 동적으로 인스톨 할 수가 있습니다.
