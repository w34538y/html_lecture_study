JNLP 파일을 이 릴리스용으로 변환하는 방법


JNLP 파일의 형식은, 1.0 으로부터 이 릴리스까지의 사이, 전혀 변경되고 있지 않습니다.

1.0 보다 낡은 버젼의 Java Web Start 에 준거한 JNLP 파일을 사용의 경우는, 파일 형식이 큰폭으로 변경되고 있으므로 주의가 필요합니다.   0.4 JNLP 파일을 Java Web Start 1.0 이상으로 준거시키기 위해서(때문에) 필요한 변경점 가운데, 가장 일반적인 것을, 다음에 열거합니다.

  • spec 속성에 1.0 을 설정한다
  • unrestricted 요소의 이름을 all-permissions 로 변경한다
  • jre 요소의 이름을 j2se 로 변경해, 그것을 resources 요소의 안쪽으로 이동한다 (즉,j2se 를 resources 의 서브 요소로 한다)

1.0 의 베타판이나 릴리스 후보로 사용되고 있던 JNLP 파일의 경우, 변환에 필요한 작업은,spec 속성을 1.0 으로 변경하는 것 뿐입니다.
