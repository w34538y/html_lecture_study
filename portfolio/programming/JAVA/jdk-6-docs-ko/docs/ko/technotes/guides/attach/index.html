Attach API

 문서의 목차
Attach API 는, Sun Microsystems 의 확장 기능으로, JavaTM 가상 머신에 첨부 메카니즘을 제공합니다. Java 언어로 쓰여진 툴은, 이 API 를 사용해 타겟의 가상 머신에 첨부해, 그 툴 에이전트를 가상 머신에 로드합니다. 예를 들어, 관리 콘솔에, 가상 머신의 계측 기구 첨부 객체로부터 관리 정보를 취득하는 관리 에이전트가 존재하면(자) 가정합니다. 관리 콘솔은, 관리 에이전트가 없는 가상 머신으로 실행하고 있는 어플리케이션을 관리할 필요가 있는 경우, 이 API 를 사용해 타겟의 가상 머신에 첨부해, 에이전트를 로드할 수 있습니다.
