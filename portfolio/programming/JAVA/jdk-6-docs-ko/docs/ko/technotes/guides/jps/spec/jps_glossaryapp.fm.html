부록 G

JavaTM 인쇄 서비스 용어집




attribute (속성)

인쇄 서비스의 기능,문서의 특성, 특정의 문서 또는인쇄 작업 전체의 처리에 대한 명령, 혹은 인쇄 작업 또는 프린터 상태에 대한 기술.

attribute category (속성 카테고리)

값의 범위를 가지지 않는속성의 이름. Java 인쇄 서비스에서는, 속성 카테고리를 OrientationRequested 클래스등의 클래스에서 표현한다. OrientationRequested 객체등의 속성 클래스의 인스턴스는,속성치이다.

attribute set (속성 세트)

속성의 집합으로, 구성되는 속성은 일반적으로, 특정의 종류의 속성인 만큼 제한된다. Java 인쇄 서비스에는, 모든 속성 세트를 위한 AttributeSet 로 불리는 슈퍼 인터페이스와 각 속성 세트를 위한 인터페이스가 정의되고 있다. 속성 세트의 종류에는,문서 속성 세트,인쇄 요구 속성 세트,인쇄 작업 속성 세트, 및인쇄 서비스 속성 세트가 있다.

attribute value (속성치)

속성이 지원하는 값의 범위에 포함되는 특정의 값. Java 인쇄 서비스로 지원되는 속성은, OrientationRequested 클래스등의 속성 클래스를 사용해 표현한다. 클래스 자체는,속성 카테고리가 된다. OrientationRequested 객체등의 클래스의 인스턴스는, 속성치가 된다. 속성치가 속성 자체를 암묵적으로 가리키기 (위해)때문에, 「속성」이라고 하는 용어가 속성치를 가리키는 것이 많다.

doc (문서)

인쇄 데이터. Java 인쇄 서비스에서는, Doc 인터페이스를 사용해 인쇄 데이터를 표현한다.

doc attribute (문서 속성)

속성의 1 개로, 개별의 문서의 특성과 그것들에 적용하는인쇄 작업의 설정을 지정한다.

doc attribute set (문서 속성 세트)

속성의 집합의 1 개로, 구성되는 속성은,문서 속성인 만큼 제한된다.

doc flavor (드큐먼트후레이바)

문서의 종류. Java 인쇄 서비스에서는, DocFlavor 클래스를 사용해, 문서의 종류를 표현한다. DocFlavor 객체에는, MIME 타입을 표현하는 String 와 Java 클래스 (java.io.InputStream 등)를 표현하는 String 가 캡슐화된다. MIME 타입은 문서의 서식을 나타내, Java 클래스 (java.io.InputStream 등)는 문서를 프린터에 송신하는 방법을 나타낸다.

hash attribute set (해시 속성 세트)

속성 세트의 1 개로, 다양한 종류의속성을 포함할 수가 있다. Java 인쇄 서비스에서는, AttributeSet 를 구현하는 HashAttributeSet 클래스를 사용해, 이 속성을 표현한다.

print job (인쇄 작업)

송신된인쇄 요구 1 개(살)로, 1 개(살) 이상의 인쇄 데이터와 처리 명령세트를 포함한다. Java 인쇄 서비스에서는, DocPrintJob 객체를 사용해 인쇄 요구를 표현한다. 이 객체는,인쇄 서비스로부터 취득한다.

print job attribute (인쇄 작업의 속성)

속성의 1 개로,인쇄 작업 상태를 보고한다. 인쇄 작업의 속성은 일반적으로, 클라이언트에서는 지정하지 않는다. 클라이언트가 인쇄 작업을 송신한 다음에, 인쇄 작업으로 지정되기 때문에 있다. 인쇄 작업 전체의 설정을 지정하려면 ,인쇄 요구의 속성을 사용한다.

print job attribute set (인쇄 작업의 속성 세트)

속성의 집합의 1 개로, 구성되는 속성은,인쇄 작업의 속성인 만큼 제한된다.

print request (인쇄 요구)

미송신의인쇄 작업 .

print request attribute (인쇄 요구의 속성)

속성의 1 개로,인쇄 작업 전체 및 인쇄 작업내의 모든 문서에 적용하는 설정을 지정한다.

print request attribute set (인쇄 요구의 속성 세트)

속성의 집합의 1 개로, 구성되는 속성은,인쇄 요구의 속성인 만큼 제한된다.

print service (인쇄 서비스)

특정의 프린터를 표현하는 인터페이스. 인쇄 작업의 팩토리.

print service attribute (인쇄 서비스의 속성)

프린터를 기술하는속성 . 예를 들어, 프린터의 장소를 지정하는 속성이 있다.

print service attribute set (인쇄 서비스의 속성 세트)

속성의 집합. 구성되는 속성은,인쇄 서비스의 속성인 만큼 제한된다.

print-service lookup (인쇄 서비스의 룩 업)

인쇄 서비스를 검색하는 처리. 즉, 지정한 처리 명령에 따라, 지정한 인쇄 데이터를 인쇄할 수 있는 프린터를 검색한다. 클라이언트는, PrintServiceLookup 클래스를 사용해 이 처리를 실행한다.

stream print service (스트림 인쇄 서비스)

클라이언트가 출력한 스트림에 데이터를 인쇄하는인쇄 서비스 .
