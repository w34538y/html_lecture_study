JAAS 튜토리얼



이 페이지로부터, JAAS (JavaTM Authentication and Authorization Service)의 다양한 사용 방법을 설명하는 2 개의 튜토리얼에 액세스 할 수 있습니다.

  1. 「JAAS 인증」

    JAAS 를 사용해, 어플리케이션으로부터 사용자의 인증을 실시하는 방법을 설명합니다.

  2. 「JAAS 승인」

    JAAS 를 사용해, 사용자 베이스의 액세스 제어를 실행하는 방법을 설명합니다.

이러한 튜토리얼로 사용하는 인증 기술은, 사용자가 특정의 이름과 패스워드를 지정하고 있을지 어떨지를 확인할 뿐(만큼)의 매우 기본적인 것입니다. 이러한 튜토리얼은, 누구라도 실행할 수 있습니다.
