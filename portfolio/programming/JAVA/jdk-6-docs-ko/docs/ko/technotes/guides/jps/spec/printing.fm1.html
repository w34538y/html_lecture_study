StreamPrintService 와 PrintService

JavaTM 인쇄 서비스 API 에는, PrintService 클래스 및 StreamPrintService 클래스가 짜넣어지고 있습니다. StreamPrintService 는 PrintService 를 상속하고 있습니다. 즉, PrintService 를 사용할 수 있을 때는, 항상 StreamPrintService 를 사용할 수 있습니다. 다만, PrintService 와 StreamPrintService 의 사용 목적은 다릅니다. PrintService 는, 출력을 프린터에 송신할 경우에 사용합니다. StreamPrintService 는, 서식 설정 끝난 인쇄 데이터를 스트림에 export 할 경우에 사용합니다. 이 때문에, 이러한 서식은 일반적으로, 다릅니다. StreamPrintService 를 검색할 경우에는, 필요한 출력의 서식을 MIME 타입의 인수로서 지정해, 데이터를 받는 OutputStream 를 지정합니다. StreamPrintService 을 검색할 때는, PrintService 를 검색할 때와 달리, 표현 클래스는 지정하지 않습니다. 출력은 항상 OutputStream 에 전달됩니다.

서비스를 검색하면(자), PrintService 또는 StreamPrintService 의 어느 쪽을 사용하는 경우에서도, 인쇄 작업을 취득해, 서비스에 송신합니다. StreamPrintService 와 PrintService 의 주된 차이는, 서비스의 검색 방법입니다. 다음의 마디에서는, 인쇄 서비스와 스트림 인쇄 서비스의 검색에 대해 설명합니다.
