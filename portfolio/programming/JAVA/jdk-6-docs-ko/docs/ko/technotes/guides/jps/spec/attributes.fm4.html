속성의 지정 방법

속성의 지정은, 데이터를 인쇄하는 경우도 스트림 전달하는 경우도, 같은 방법으로 실시합니다. 일반적으로은, PrintRequestAttribute 를 사용해 속성을 지정합니다. 인쇄 요구 속성에는, 인쇄 작업 전체의 설정을 지정하기 (위해)때문에입니다. 특정의 작업에 대해서 복수의 속성을 지정하는 것이 많기 때문에, 속성 세트를 작성할 필요가 있습니다. 다음의 예에서는, PrintRequestAttributeSet 를 작성해, A4 매체를 사용해 작업을 5 부 양면 인쇄하고 있습니다. 

PrintRequestAttributeSet aset = new HashPrintRequestAttributeSet();
aset.add(new Copies(5)); 
aset.add(MediaSize.ISO_A4); 
aset.add(Sides.DUPLEX);
속성 세트를 작성한 다음에, DocFlavor 와 함께 인쇄 작업의 인쇄 메소드에 속성 세트를 건네줍니다. DocFlavor 에 대해서는,「문서의 종류의 지정」을 참조해 주세요. 인쇄 방법에 대해서는,「문서의 인쇄 및 스트리밍」을 참조해 주세요. 다음의 마디에서는, 표준 속성에 대해 상세하게 설명합니다.
