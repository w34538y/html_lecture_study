10 사례

Java 2 SDK 의 새로운 시큐리티 기능의 설계 및 구현의 대부분은, Java Software 시큐리티 그룹의 멤버에 의해 행해졌습니다. 그 이외의 (과거 및 현재의) Java Software의 멤버도 헤아릴 수 없을 정도(수록) 귀중한 통찰, 상세한 리뷰, 및 필요한 기술적 지원을 주어주셨습니다. 막대한 공헌을 해 주신 것은 이하의 분들입니다 (알파벳순서. 다만, 여기서 모든 분들의 이름을 들 수 없습니다). Gigi Ankeny, Josh Bloch, Satya Dodda, Charlie Lai, Sheng Liang, Jan Luehe, Marianne Mueller, Jeff Nisewanger, Hemma Prafullchandra, Roger Riggs, Nakul Saraiya, Bill Shannon, Roland Schemers, 및 Vijay Srinivasan

이번 프로젝트는, Java Software 의 관리자 (Dick Neiss, Jon Kannegaard, 및 Alan Baratz 에 감사합니다), 실험 그룹, 및 문서 작성 그룹 (특히 Mary Dageforde)의 강력한 지원없이는 실현될 수 없었습니다. James Gosling, Graham Hamilton, 및 Jim Mitchell 로부터의 기술적인 어드바이스에도 진심으로 감사합니다.

여기에는 이름을 들 수가 없습니다만, 폐사의 기업 파트너 및 라이센스 보관 유지자의 여러분으로부터도 많은 제안을 받았습니다.
