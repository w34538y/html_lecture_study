슈퍼 클래스가 외부화 가능하지 않은 경우의 외부화

예제의 실행

이 예제를 컴파일 해 실행하려면 , 다음의 순서에 따릅니다.

  1. 자신의 베이스 디렉토리아래의 objectserializationexamples/NonExternalizableSuperclass 디렉토리에서, JDKTM 1.1 이후를 사용해 Nonexternsuper.java 를 컴파일 합니다. 
      javac Nonexternsuper.java
  2. 컴파일 한 파일을 실행합니다. 
      java Nonexternsuper

직렬화전과 직렬화 후에 Book 객체가 출력됩니다.
