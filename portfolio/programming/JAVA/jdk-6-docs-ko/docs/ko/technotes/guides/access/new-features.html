JavaTM 2 플랫폼 v1. 4 의 신기능

 Java Accessibility
여기서 설명하는 기능은, Java 2 Platform v1. 4.0 으로 도입되었습니다. 이러한 기능도, Java 2 플랫폼의 향후의 릴리스에 짜넣어집니다.

AccessibleExtendedComponent.getAccessibleKeyBinding
AccessibleExtendedComponent 인터페이스에 새로운 메소드 getAccessibleKeyBinding 가 추가되었습니다. 이 메소드는, 객체에 관련지을 수 있고 있는 키의 할당을 돌려줍니다.
AccessibleRole 의 새로운 정수
AccessibleRole 클래스에, 다음의 새로운 정수가 추가되었습니다.
  • DATE_EDITOR
  • FONT_CHOOSER
  • GROUP_BOX
  • SPIN_BOX
  • STATUS_BAR
HTML Object 태그에 대한 Accessibility 지원
이 릴리스에서는, HTML 의 object 태그에 대한 사용자 보조 기능이 추가되고 있습니다. Ctrl + t 키와 Shift + Ctrl + t 키를 사용해, HTML 문서내의 object 태그에 관련지을 수 있고 있는 다음의 컴퍼넌트 및 전의 컴퍼넌트에 각각 이동할 수 있습니다. Ctrl + Space 키를 누르면(자), 컴퍼넌트에 관련지을 수 있고 있는 디폴트의 액션이 기동됩니다.
HTMLEditorKit 키보드 링크 traversal의 추적 지원
사용자 보조 기능에는, HTMLEditorKit 키보드 링크 traversal를 추적하는 수단이 필요합니다. 이 기능을 지원하기 위해서, 정수 AccessibleContext.ACCESSIBLE_HYPERTEXT_OFFSET 가 추가되었습니다.
사용자 보조 기능의 프로퍼티
javax.accessibility.assistive_technologies 프로퍼티은, 사용자 보조 기능을 JVM 에 로드하도록(듯이) 지정합니다.
화면 리더의 프로퍼티
javax.accessibility.screen_reader_present 프로퍼티이 true 로 설정되어 있는 경우, Java 플랫폼 라이브러리는 화면 리더가 시스템에 존재한다고 인식합니다. 어플리케이션 개발자는 이 프로퍼티을 확인할 수 있습니다. 이 프로퍼티을 체크하면(자), 자기 음성을 사용하는 어플리케이션에서는, 화면 리더가 존재하는 경우는 자기 음성을 오프로 할 수 있습니다.
API 의 확장 기능
JavaTM Accessibility API 에 다음의 클래스 및 인터페이스가 추가되었습니다.
  • javax.accessibility.AccessibleEditableText
  • javax.accessibility.AccessibleExtendedComponent
  • javax.accessibility.AccessibleExtendedTable
  • javax.accessibility.AccessibleKeyBinding
또,javax.accessibility.AccessibleContext 가 갱신되어 새로운 클래스 및 인터페이스가 반영되었습니다.
AccessibleEditableText
AccessibleEditableText 인터페이스는, 사용자 보조 기술에 의한 텍스트 편집을 지원합니다.
AccessibleExtendedComponent
AccessibleExtendedComponent 인터페이스는, 객체의 그래피컬한 표현을 결정 및 설정하는 사용자 보조 기능 (또는 기술)의 표준적인 기구를 제공합니다.
AccessibleExtendedTable
AccessibleExtendedTable 인터페이스는, 2 차원 테이블 형식에서 데이터를 나타내는 사용자 인터페이스 컴퍼넌트의 상세 정보를 나타냅니다.
AccessibleKeyBinding
AccessibleKeyBinding 인터페이스는, 객체용의 키의 할당을 판별하기 위한 , 사용자 보조 기술용의 표준적인 기구를 제공합니다.
니마닉(mnemonic)를 이용한 탭 네비게이션
이 신기능에서는,JTabbedPane 로 니마닉(mnemonic)를 사용하는 것으로써, 한층 용이한 탭 네비게이션이 지원되고 있습니다.
최초의 문자에 의한 네비게이션의 일람표시
이 신기능에서는, 리스트 항목의 최초의 문자를 사용해 네비게이션을 일람표시 하는 기능이 지원되고 있습니다.
HTML 컴퍼넌트용 Accessibility 지원
이 릴리스에서는, Swing 의 HTML 컴퍼넌트에 대해서 사용자 보조 기능이 제공됩니다.
화면 확대 기능의 프로퍼티
javax.accessibility.screen_magnifier_present 프로퍼티이 true 로 설정되어 있는 경우, Java 플랫폼 라이브러리는 화면 확대 기능이 시스템에 존재한다고 인식합니다. 어플리케이션 개발자는 이 프로퍼티을 체크해, 화면 확대 기능이 존재하는 경우에는, 어플리케이션이 화면 확대 기능에 대응하고 있는 것을 확인할 필요가 있습니다.
