Java 2DTM API 스펙

 문서의 목차
이하의 리스트는, Java 2D API 에 짜넣어지고 있는 패키지와 클래스의 일람입니다. 리스트의 각 항목은, 각각의 API 문서에 링크되고 있습니다.
  • java.awt.color
    ICC 프로파일 형식 스펙에 근거하는, 칼라 영역과 칼라 프로파일 데이터의 클래스를 제공한다
  • java.awt.font
    폰트에 관련하는 클래스와 인터페이스, 및 Type 1, Type 1 Multiple Master, OpenType, 및 TrueType 폰트를 나타내기 위한 지원를 제공한다
  • java.awt.geom
    2 차원의 지오 시학 객체의 정의 및 조작을 행하기 위한 클래스를 제공한다
  • java.awt.image
    이미지의 작성이나 변경을 실시하는 클래스를 제공한다. 또, 이미지 처리를 구성하기 위한 , 이메이지프로듀서, 컨슈머(consumer), 및 필터를 제공한다
  • java.awt.print
    2 차원 객체를 인쇄하기 위한 클래스와 인터페이스를 제공한다. 또, 문서의 종류를 지정할 수가 있어 페이지 설정과 페이지 서식의 제어 및 작업 제어 다이얼로그의 관리도 실시한다
  • javax.print
    Java 인쇄 서비스 API 의 주체가 되는 클래스와 인터페이스를 제공한다
  • java.awt.image.renderable
    draw에 의존하지 않는 이미지를 작성하기 위한 클래스 및 인터페이스를 제공한다

java.awt 패키지에는, Java 2D API 에 포함되는 클래스도 짜넣어지고 있습니다.

  • java.awt - 사용자 인터페이스의 생성 및 그래픽스와 이미지의 인쇄를 실시하는 클래스가 포함됩니다. 이하의 java.awt 클래스는, Java 2D API 의 일부이다
