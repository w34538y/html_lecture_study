시큐리티

 문서의 목차

Java™ 시큐리티 테크놀러지에는, 많은 API, 툴, 일반적으로 사용하는 시큐리티 알고리즘, 기구, 프로토콜의 구현이 포함되어 있습니다. Java 시큐리티 API 는, 암호화, 공개키 인프라스트럭쳐(infrastructure), 안전한 통신, 인증, 액세스 제어 등, 광범위한 영역에 이르고 있습니다. Java 시큐리티 테크놀러지는, 개발자에게 어플리케이션을 기술하기 위한 포괄적인 시큐리티 시스템를 제공해, 사용자 또는 관리자에게 어플리케이션을 안전하게 관리하기 위한 일련의 툴을 제공합니다.
