J2SE 5.0 에 있어서의 JavaBeansTM 컴퍼넌트 API 에 대한 API 의 확장 기능

 JavaBeansTM Component API
이 문서에서는, JavaTM 2 Platform, Standard Edition 버젼 5.0 으로 java.beans 패키지에 더해진 변경점에 대해 설명합니다.
  • IndexedPropertyChangeEvent 의 지원
  • PropertyEditor 작성의 새로운 방법
  • PropertyEditorSupport 의 public 생성자

IndexedPropertyChangeEvent 의 지원

이 변경에 관련하는 버그 추적 리포트: 4353056

IndexedPropertyChangeEvent 라고 명명해진 PropertyChangeEvent 의 새로운 서브 클래스가, 변경된 Bean 의 일부를 인식하기 위해서 색인을 사용하는, 지원 바운드 프로퍼티에 추가되었습니다. 또, 모두 fireIndexedPropertyChange 라고 명명해진 3 개의 메소드가,PropertyChangeSupport 유틸리티 클래스에 추가되어 인덱스 프로퍼티의 변경 이벤트의 발생을 지원합니다. 인덱스 프로퍼티의 변경 이벤트를 사용하는 예에 대해서는,IndexedTest.java 를 참조해 주세요.


PropertyEditor 작성의 새로운 방법

이 변경에 관련하는 버그 추적 리포트: 4274639

createPropertyEditor 로 불리는 새로운 메소드가,PropertyDescriptor 클래스에 추가되었습니다.


PropertyEditorSupport 의 public 생성자

이 변경에 관련하는 버그 추적 리포트: 4305280

2 개의 PropertyEditorSupport 생성자 은 protected 생성자 은 아니고 public 생성자 이 되었습니다. 그것에 의해 PropertyEditorSupport 객체는 보다 용이하게 작성해, 위양으로서 사용할 수 있습니다. 또, 소스 객체는, 새로운 setSource 및getSource 메소드에 의해 공개되게 되었습니다.
