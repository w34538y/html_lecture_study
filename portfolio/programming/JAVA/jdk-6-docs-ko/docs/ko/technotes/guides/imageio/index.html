JavaTM Image I/O 테크놀러지

 문서의 목차
JavaTM Image I/O API 는,JSR-015 에 의해 개발된 API 로, 파일에 포함된 이미지나, 네트워크 경유로 액세스 하는 이미지를 처리하기 위한 , 플러그 인 가능한 아키텍쳐(architecture)를 제공합니다. 이 API 는, 이미지를 로드하거나 보존하거나하기 위한 이전의 API 에 비해, 실질적으로 유연성이 높고, 기능도 강력합니다.

이 API 는, 이미지의 형식에 특유의 플러그 인을 추가하기 위한 시스템를 제공합니다. 자주(잘) 사용되는 몇개의 이미지 형식용의 플러그 인이 Java Image I/O 에 짜넣어지고 있습니다만, 서드 파티가 특수한 이미지 형식용의 독자적인 플러그 인을 작성하기 위해서 이 API 를 이용할 수도 있습니다.

API 의 스펙

Java Image I/O API 스펙은, 다음의 패키지로 구성되어 있습니다.

  • javax.imageio
    이미지 파일의 내용을 기술하기 위한 기본적인 클래스와 인터페이스가 포함되어 있습니다. 메타데이타나 엄지손가락 썸네일의 짜넣어, 이미지를 읽고 쓰기하는 프로세스의 제어, 이미지 형식간의 변환의 실행, 및 에러의 보고가 가능합니다.
  • javax.imageio.event
    이미지의 읽고 쓰기 처리동안에 발생하는 이벤트를 동기적으로 통지하기 위한 인터페이스가 포함되어 있습니다.
  • javax.imageio.metadata
    이미지의 메타데이타를 읽고 쓰기하기 위한 클래스와 인터페이스가 포함되어 있습니다.
  • javax.imageio.plugins.bmp
    편입의 BMP read 플러그 인 및 기입해 플러그 인을 위한 지원 클래스가 포함되어 있습니다.
  • javax.imageio.plugins.jpeg
    편입의 JPEG read 플러그 인 및 기입해 플러그 인을 위한, 몇개의 지원 클래스가 포함되어 있습니다.
  • javax.imageio.spi
    read 플러그 인, 기입 플러그 인, 변환 플러그 인을 위한 서비스 프로바이더 인터페이스가 포함되어 있습니다. 또, 실행시 레지스트리도 제공합니다.
  • javax.imageio.stream
    파일 및 스트림로부터의 저레벨의 입출력을 제공하는 클래스와 인터페이스가 포함되어 있습니다.

튜토리얼과 개발자 안내

  • 「Java Image I/O API 가이드」 에서는, Java Image I/O API 의 상세와 Java Image I/O 를 사용해 이미지를 읽고 쓰기하는 방법이나 커스텀 플러그 인의 작성 방법을 기재하고 있습니다.
