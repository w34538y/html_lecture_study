부록 B:sound.properties 파일

Java Sound 설정 파일 sound.properties 는,javax.sound.sampled.AudioSystem 와 javax.sound.midi.MidiSystem 의 디폴트의 서비스 프로바이더를 지정하기 위해서 사용합니다.

프로퍼티 키

다음의 키는,AudioSystem 메소드에 의해 인식됩니다.

javax.sound.sampled.Clip
javax.sound.sampled.Port
javax.sound.sampled.SourceDataLine
javax.sound.sampled.TargetDataLine

다음의 키는,MidiSystem 메소드에 의해 인식됩니다.

javax.sound.midi.Sequencer
javax.sound.midi.Synthesizer
javax.sound.midi.Transmitter
javax.sound.midi.Receiver

이러한 값에는, 서비스 프로바이더의 완전한 클래스명 또는 디바이스명 (혹은 그 양쪽 모두)을 지정합니다.

Sun 의 구현에서는,sound.properties 는 $JAVAHOME/jre/lib 에 있습니다.

자세한 것은,AudioSystem 와 MidiSystem 의 클래스의 설명을 참조해 주세요.

례

MyDeviceProvider 를 SourceDataLine 라인의 디폴트로서 사용하려면 , 다음의 키를 설정합니다.

javax.sound.sampled.SourceDataLine=com.xyz.MyDeviceProvider

디폴트의 Synthesizer 를 그 이름 InternalSynth 에 의해 지정하려면 , 다음의 키를 설정합니다.

javax.sound.midi.Synthesizer=#InternalSynth

디폴트의 Receiver 를 프로바이더와 이름에 의해 지정하려면 , 다음의 키를 설정합니다.

javax.sound.midi.Receiver=com.sun.media.sound.MidiProvider#SunMIDI1
