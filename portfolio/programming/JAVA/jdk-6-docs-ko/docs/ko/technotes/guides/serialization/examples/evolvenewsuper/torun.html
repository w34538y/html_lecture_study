직렬화, 전개, 및 새로운 슈퍼 클래스의 추가

예제의 실행

이 예제를 컴파일 해 실행하려면 , 다음의 순서에 따릅니다.

  1. 자신의 베이스 디렉토리아래의 objectserializationexamples/AddingNewSuperClassAndEvolution 디렉토리에서, JDKTM 1.2 이후를 사용해 오리지날 클래스를 컴파일 합니다. 
      javac OriginalClass.java
  2. 직렬화 플래그를 지정해, 오리지날 클래스를 실행합니다. 
      java OriginalClass -s
  3. 전개 후의 클래스를 컴파일 합니다. 
      javac EvolvedClass.java
  4. 직렬화 복원 플래그를 지정해, 전개 후의 클래스를 실행합니다. 
      java EvolvedClass -d

이 방법에서는, 한방향만의 호환성이 테스트됩니다. 쌍방향의 호환성을 테스트하려면 , 같은 순서를 역방향으로 행해 주세요 (순서 1 에서 4 로, 오리지날 클래스명과 전개 후의 클래스명을 바꿔 넣는다).
