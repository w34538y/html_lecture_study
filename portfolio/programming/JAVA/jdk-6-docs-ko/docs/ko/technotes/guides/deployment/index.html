JavaTM 의 배치

 문서의 목차

Java SE 6 에 있어서의 Java 배치의 신기능이나 확장 기능점에 대해서는,「Deployment Enhancements in Java SE 6」을 참조해 주세요.

JavaTM 배치 에서는, 폭넓은 내용을 취급합니다. 다음의 내용이 있습니다.

  • 컴퓨터에의 Java 플랫폼의 인스톨
  • Java 컨트롤 패널의 설정 옵션
  • Java 프로그램 언어에서의 어플리케이션이나 애플릿의 작성
  • 애플릿을 호출하거나 어플리케이션을 다운로드해 기동하거나하기 위한 Web 페이지의 작성
  • Web 서버로 Java 관련 파일을 이용 가능하게 한다
  • 컴퓨터상의 Java 플랫폼의 갱신

이러한 내용중 Java 컨트롤 패널의 설정 옵션이나 컴퓨터상의 Java 플랫폼의 코우신에 도착해, 지금부터 설명하는 「Java 배치 가이드」로 설명합니다. Java 플랫폼의 인스톨이나 어플리케이션 및 애플릿의 작성 등, 그 외의 내용에 대해서는 다른 문서로 취급합니다. 상기의 토픽에 대한 자세한 것은, 「Java 배치 가이드」의「Java 배치의 개요」를 참조해 주세요.

Java 배치 가이드

「Java 배치 가이드」에서는,Java 배치의 개요, Java 컨트롤 패널, Java Update 기구, Microsoft 와 Sun VM 에 관한 애플릿 호환성, Windows 에의 사일런트 인스톨, 자동 다운로드, 인스톨, 갱신으로 자주 있는 질문등을 설명합니다.
