옵션 패키지의 지원

 문서의 목차

옵션 패키지는, 클래스와 거기에 관련하는 native code의 패키지입니다. 확장 기능을 사용하는 것으로, 어플리케이션의 개발자는 코어 플랫폼의 기능을 확장할 수 있습니다. 확장 기능 기구에 의해, Java 플랫폼을 사용하는 경우와 같이, Java 가상 머신으로부터 옵션의 확장 기능 클래스를 이용할 수 있습니다. 확장 기능 기구에 의해, 필요한 옵션 패키지가 JDK 또는 JRE 에 인스톨되어 있지 않은 경우는, 지정된 URL 로부터 취득할 수도 있습니다.

개요
  • 개요 - 옵션 패키지의 개요 및 사용 방법에 대해 설명합니다.
API 의 스펙

확장 기능 기구에서는, 다음의 클래스를 이용합니다.

  • java.lang.ClassLoader
  • java.lang.Package
  • java.lang.Thread
  • java.net.JarURLConnection
  • java.net.URLClassLoader
  • java.security.SecureClassLoader
이러한 API 의 설명에 대해서는, 확장 기능 기구의 스펙의「API 의 상세」도 참조해 주세요.

API 의 확장 기능

  • 버젼 1.3 의 확장 기능 기구에서는, Manifest 속성이 확장되고 있습니다. 확장된 Manifest 속성을 사용해, 인스톨 되고 있는 옵션 패키지의 벤더 및 버젼 관리 정보를 확인할 수 있습니다. 애플릿에 필요한 옵션 패키지가 인스톨되어 있지 않은 경우, 부정한 버젼이 인스톨 되고 있는 경우, 또는 적절한 벤더의 옵션 패키지가 아닌 경우는, Java Plug-in 를 사용해, 지정된 URL 로부터 필요한 확장 기능을 다운로드할 수 있습니다. 자세한 것은,「옵션 패키지의 버젼 관리」 를 참조해 주세요.
