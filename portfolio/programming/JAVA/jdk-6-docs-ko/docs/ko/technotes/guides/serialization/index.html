객체 직렬화

 문서의 목차  
객체 직렬화는, 객체와 거기로부터 참조되고 있는 객체를 바이트 스트림에 코드화합니다. 또, 그 스트림로부터 객체 그래프의 보완적인 재구축을 실시합니다. 직렬화는, 간단하고 쉬운 지속성의 실현이나, 소켓이나 Java Remote Method Invocation (Java RMI)에 의한 통신을 위해서(때문에) 사용됩니다. 객체의 디폴트의 코드화에서는, private 와 transient 의 데이터가 보호되어 클래스의 전개가 지원됩니다. 클래스는, 독자적인 코드화를 구현할 수가 있기 (위해)때문에, 외부 포맷에 대해서는 단독으로 책임을 가집니다.

새롭고, 객체의 직렬화 된 데이터를, 클래스의 필드 마다 개별적으로 지정하기 위한 API 가 추가되었습니다. 또, 기존의 프로토콜을 사용해, 직렬화 된 데이터 필드를 스트림로부터 읽어들이거나 스트림에 기입하는 것으로, 디폴트의 읽어들여/기입 기구와의 호환성을 보증하는 일도 가능하게 되었습니다.
