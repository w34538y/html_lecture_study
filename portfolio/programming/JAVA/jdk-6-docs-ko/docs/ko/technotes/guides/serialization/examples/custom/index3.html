커스텀 데이터 형식과 직렬화의 사용

개설

이 예제는,writeObject 메소드와 readObject 메소드를 사용해 커스텀 데이터 형식에 코드화하는 방법을 나타냅니다. 대량의 지속성 데이터가 있는 경우는, 이 예제로 소개하도록(듯이), 데이터를 압축 형식에서 보존하는 편이 편리합니다.

이 예제에서는 삼각 배열을 사용합니다. 삼각 배열은, 대칭형의 이차원 배열입니다. 직렬화에서는, 배열의 반만큼이 보존됩니다. 이 예제에서는, 배열의 반을 writeObject 에 직렬화해, 다음에 그 반을 직렬화 복원해, 한층 더 카피해 나머지의 반으로서 복원하는 것으로, 배열 전체를 readObject 에 복원합니다.

이 예제는, 버젼 관리가 지원되지 않는다고 하는 점으로써,「직렬화와 직렬화 가능 필드 API 의 사용」의 예제와는 다릅니다. 게다가 이 예제에서는 Externalizable 인터페이스를 사용하지 않기 때문에, 슈퍼 클래스는 고려되고 있지 않습니다.
