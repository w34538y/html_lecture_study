JMX 테크놀러지의 버젼

Java 2 Platform Standard Edition (J2SE)로 채용된 JMX 테크놀러지는, 긴밀히 관련한 2 개의 Java Specification Requests (JSR)로서 Java Community Process (JCP)에 의해 개발되었습니다. J2SE 플랫폼의 버젼 5.0 에 구현되고 있는 JSR 의 버젼에 대해, 이하의 섹션으로 설명합니다.

Java Management Extensions Instrumentation and Agent Specification (JSR 3)

Java SE 6 플랫폼은, 「JMX 스펙」의 버젼 1.4 (보수 릴리스, 2006 년 10 월)를 구현하고 있습니다. 다운로드로 제공되고 있던 정오표에 있던 수정이 포함되어 있습니다.

JMX Remote API Specification (JSR 160)

Java SE 6 플랫폼은, 「JMX Remote API 의 스펙」의 버젼 1.4 (보수 릴리스, 2006 년 10 월)를 구현하고 있습니다.

JSR 160 에서는, 표준의 RMI 및 RMI/IIOP 연결기 외에, TCP 소켓을 기본으로 한, 생략 가능한 JMX Messaging Protocol (JMXMP)도 정의하고 있습니다. Java SE 플랫폼에 이러한 생략 가능한 연결기는 포함되어 있지 않습니다. 다만, 보다 고도의 레벨의 시큐리티를 필요로 하는 경우등으로는, JMXMP 연결기를 사용할 수 있습니다.

JMXMP 연결기를 사용하는 경우는, JSR 160 레퍼런스 구현을 이 섹션의 첫머리에서 지정한 다운로드 페이지로부터 다운로드해,jmxremote_optional.jar 파일을 classpath 에 추가해 주세요. JSR 160 레퍼런스 구현에 포함되는 「JMX Remote API Tutorial」에는, JMXMP 연결기의 사용예가 있습니다.
