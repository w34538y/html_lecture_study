JavaTM 2 플랫폼의 시큐리티 아키텍쳐(architecture)




저자 :Li Gong

버젼 1.2

이 문서에서는, JavaTM 2 플랫폼의 시큐리티에 구현 예정의 주요한 시큐리티 기능의 설계 동기에 대해 설명해, 새로운 클래스와 그 사용법을 소개합니다. 또, 이 새로운 아키텍쳐(architecture)의 기존의 코드에의 영향에 대해 논해 시큐리티를 고려한 코드의 기술에 관한 고찰을 실시합니다.
