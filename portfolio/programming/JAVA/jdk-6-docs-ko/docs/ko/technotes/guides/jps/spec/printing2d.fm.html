제 5 장

2D 그래픽스의 인쇄와 스트림 전달




Java 2DTM 인쇄 API 는, java.awt.print 패키지로, JavaTM 2 SE, version 1.2 이후에 짜넣어지고 있습니다. Java 2D 인쇄 API 를 사용해, PrinterJob 를 작성하거나 사용자용의 프린터 다이얼로그를 표시하거나 화면에 draw 할 경우에 사용되는 java.awt.Graphics 및 java.awt.Graphics2D 클래스를 사용해, 그래픽스를 페이지 분할한 인쇄 이바지하거나 할 수 있습니다.

Java 인쇄 서비스에서의 신기능의 상당수는, 프린터의 검출이나 인쇄 속성의 지정 등, Java 2D 인쇄 API 사용자에 있어서도 중요한 기능입니다. Java 2D 인쇄의 사용자가 이러한 기능을 사용하려면 , java.awt.print 패키지를 JavaTM 2 SE, version 1.4 전용으로 갱신해, Java 2D 인쇄 API 로부터 JavaTM 인쇄 서비스에 액세스 할 필요가 있습니다.

Java 2D 인쇄 어플리케이션의 개발자는, 다음의 4 개의 방법에 따라, Java 인쇄 서비스와 Java 2D API 를 사용할 수 있습니다.

  • PrinterJob 를 사용해, 2D 그래픽스를 인쇄한다
  • PrinterJob 를 사용해, 2D 그래픽스를 스트림 전달한다
  • DocPrintJob 와 서비스 서식이 끝난 DocFlavor 를 사용해, 2D 그래픽스를 인쇄한다
  • DocPrintJob 와 서비스 서식 설정이 끝난 DocFlavor 를 사용해, 2D 그래픽스를 스트림 전달한다
「 PrinterJob 를 사용한 그래픽스의 인쇄 또는 스트림 전달」 에서는, 최초의 2 개의 방법에 대해 설명합니다. 「서비스 서식 설정이 끝난 데이터의 사용」에서는, 마지막 2 개의 방법에 대해 설명합니다.
