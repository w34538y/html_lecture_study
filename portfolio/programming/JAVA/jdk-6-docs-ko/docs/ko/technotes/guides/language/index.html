Java 프로그램 언어

 문서의 목차
Java 프로그램 언어는, 동시 처리가 가능해, 강하고 형지를 무늬를 박아서 염색한 것 된다고 하는 특징을 갖추어 클래스를 베이스로 한, 범용의 객체 지향 언어입니다. 일반적으로은, Java 가상 머신 스펙으로 정의된 바이트 코드 명령 세트 및 바이너리 형식에 컴파일 됩니다.
