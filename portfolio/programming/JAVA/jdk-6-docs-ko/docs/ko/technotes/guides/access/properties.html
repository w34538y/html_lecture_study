Accessibility 의 프로퍼티

 Java Accessibility

이용 가능한 프로퍼티

javax.accessibility 패키지에 의해, 다음의 프로퍼티이 제공됩니다.
  • assistive_technologies
  • screen_magnifier_present
  • screen_reader_present

사용자 보조 기능의 로드

assistive_technologies 프로퍼티은, 사용자 보조 기능을 JVM 에 로드하도록(듯이) 지정합니다. 그 입력에는, 콤마로 단락지어진 리스트가 사용됩니다.

화면 확대 기능의 존재의 표시

screen_magnifier_present 프로퍼티이 true 로 설정되어 있는 경우, Java 플랫폼 라이브러리는 화면 확대 기능이 시스템에 존재하는 것을 인식합니다. 어플리케이션 개발자는 이 프로퍼티을 체크해, 화면 확대 기능이 존재하는 경우에는, 어플리케이션이 화면 확대 기능에 대응하고 있는 것을 확인할 필요가 있습니다. 예를 들어, Microsoft Windows operating system에서는, Java 2DTM API 의 레퍼런스 구현이 이 프로퍼티을 체크해,true 의 경우에는 화면 확대 기능으로 문제가 발생하지 않게 Microsoft DirectDraw 를 오프로 합니다. (화면 확대 기능에서는, DirectDraw 를 이용한 그래픽스를 확대할 수 없는 경우가 있습니다. )

화면 리더의 존재의 표시

screen_reader_present 프로퍼티이 true 로 설정되어 있는 경우, Java 플랫폼 라이브러리는 화면 리더가 시스템에 존재하는 것을 인식합니다. 어플리케이션 개발자는 이 프로퍼티을 확인할 수 있습니다. 이 프로퍼티을 체크하면(자), 자기 음성을 사용하는 어플리케이션에서는, 화면 리더가 존재하는 경우는 자기 음성을 오프로 할 수 있습니다.

프로퍼티의 설정

프로퍼티은, 실행시로 설정하는지, 프로퍼티 파일내에서 설정할 수 있습니다. 프로퍼티을 실행시로 설정하는 경우는, 다음과 같은 커멘드를 입력합니다.

java -Djavax.accessibility.screen_magnifier_present=true foo.class

레퍼런스 구현에는,jre/lib 디렉토리내의 accessibility.properties 파일이 포함됩니다. 프로퍼티을 accessibility.properties 파일내에서 설정하는 경우는, 다음과 같은 행을 추가합니다.

assistive_technologies=Ferret, Monkey
screen_magnifier_present=true
screen_reader_present=true
