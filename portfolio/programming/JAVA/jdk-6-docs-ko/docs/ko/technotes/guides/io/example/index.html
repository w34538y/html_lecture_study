NIO 의 예

 문서의 목차
NIO
다음에 드는 예는, NIO API 의 사용법을 나타내고 있습니다.

원시 파일 설명
Grep.java 파일 리스트로부터, 지정된 정규 표현 패턴에 매치 하는 행을 검색한다. NIO 맵 바이트 버퍼, 캐릭터 세트, 및 정규 표현을 예시한다
Sum.java 파일 리스트의 16 비트의 체크 섬을 계산한다. 고속화를 위해, NIO 맵 바이트 버퍼를 사용한다
TimeQuery.java 호스트의 리스트에 시각을 조회한다. NIO 소켓 채널 (접속과 읽어내), 버퍼의 취급해, 캐릭터 세트, 및 정규 표현을 예시하는, 간단한 블록 프로그램
TimeServer.java 접속을 대기해, 호출 측에 시각을 돌려준다. NIO 소켓 채널 (수락과 기입해), 버퍼의 취급해, 캐릭터 세트, 및 정규 표현을 예시하는, 간단한 블록 프로그램
Ping.java 호스트의 각 리스트에 접속해, 접속을 완료하는데 필요로 하는 시간을 계측 한다. 셀렉터와 2 개의 부가적인 thread를 사용해, 비블록 접속과 셀렉터의 multi-thread화한 사용을 예시한다
NBTimeServer.java 비블록의 인터넷 시각 서버를 구현한다
