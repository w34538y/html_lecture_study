Abstract Window Toolkit (AWT)

 문서의 목차

Abstract Window Toolkit (AWT)는, 그래피컬 사용자 인터페이스 (GUI) 프로그램의 작성을 지원합니다. AWT 에는 다음의 특징이 있습니다.
  • 네이티브인 사용자 인터페이스 컴퍼넌트세트
  • 견뢰한 이벤트 처리 모델
  • 도형, 색, 폰트의 클래스를 포함한, 그래픽스 및 화상 처리용 툴
  • 특정의 화상 사이즈나 해상도에 의존하지 않는 유연한 윈도우 레이아웃을 가능하게 하는 레이아웃 매니저
  • 네이티브 플랫폼의 클립보드를 통한 컷＆페이스트를 가능하게 하는 데이터 전송 클래스
Swing 클래스가 AWT 아키텍쳐(architecture)의 최상부에 구축되었습니다.
