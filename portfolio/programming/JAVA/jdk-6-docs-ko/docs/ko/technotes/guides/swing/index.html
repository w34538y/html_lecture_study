Swing
(JavaTM Foundation Classes)

 문서의 목차
Swing 클래스는 JavaTM Foundation Classes (JFC) 소프트웨어의 일부로, 그래피컬 사용자 인터페이스 (GUI)를 구축해, 풍부한 그래픽 기능 및 쌍방향성을 Java 어플리케이션에 추가하는 컴퍼넌트세트를 구현하고 있습니다. Swing 컴퍼넌트는, 전체가 Java 프로그램 언어에 의해 구현되고 있습니다. 플러그 인 가능한 Look & Feel 에 의해, 플랫폼간에 같은 외관이 되는 GUI 나, 현재의 OS 플랫폼 (Microsoft Windows, SolarisTM, Linux 등)의 Look & Feel 를 상정한 GUI 를 작성할 수 있습니다.
