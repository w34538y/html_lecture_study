Java I/O

문서의 목차

Java I/O 의 지원는,java.io 및 java.nio 패키지에 포함되어 있습니다. 이러한 패키지에는 다음의 기능이 부속됩니다.

  • 파일 및 디바이스의 입출력
  • 객체 직렬화
  • 원시형의 데이터용의 버퍼
  • 캐릭터 세트의 엔코더 및 디코더
  • 채널 (새로운 프리미티브(primitive) 입출력의 추상화)
  • 락과 메모리맙핑을 지원하는 파일 인터페이스
  • 측정할 수 있는 서버에 출력하기 위한 , 다중화 된 비블록 입출력 기능

API 의 스펙

  • java.io (설명) - 시스템의 입출력, 객체 직렬화, 파일 시스템에의 기본적 액세스를 지원
  • java.nio (설명) - 바르크메모리를 조작하기 위한 「버퍼」를 정의. 퍼포먼스 향상을 위해, 버퍼를 직접 메모리에 할당할 수가 있습니다.
  • java.nio.channels (설명) - 「채널」을 정의해, 입출력을 실행 가능한 디바이스를 추상화. 다중화 된 비블록 입출력을 위한 「셀렉터」를 정의합니다.
  • java.nio.channels.spi (설명) - 채널에 구현을 제공
  • java.nio.charset (설명) - 바이트와 Unicode 문자와의 변환을 위해, 캐릭터 세트, 디코더, 엔코더를 정의
  • java.nio.channels.spi (설명) - 캐릭터 세트에 구현을 제공
