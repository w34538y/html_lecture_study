1.4 대상 독자

이 API 는, 다양한 문맥으로 이용되는 것을 의도하고 있습니다. 예를 들어, 클라이언트측의 어플리케이션으로 이 API 를 사용해, 로컬 디스크로부터, 또는 네트워크 경유로, 이미지 데이터를 로드할 수가 있습니다. 또, 예를 들어, Servlet API 를 사용하는 등, 서버측의 어플리케이션으로 이 API 를 사용해, 클라이언트로부터의 요구에 응해 동적으로 이미지를 생성할 수도 있습니다. 게다가 Image I/O API 는, Java Advanced Imaging, Java3D, Java Media Framework 등, 다른 API 와 함께 이용해, 선진적인 멀티미디어 어플리케이션을 구축할 수도 있습니다.
