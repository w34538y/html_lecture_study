Java 인쇄 서비스 API 의 특징

Java 인쇄 서비스 API 에서는, Java 플랫폼상에서의 인쇄를 통합 및 확장하기 위해서, 클라이언트 및 서버 어플리케이션에 존재하는 많은 공통 인쇄 요건 가운데, 현재의 Java 인쇄 API 로 해결되어 있지 않은 것에 대응하고 있습니다.

이 Java 인쇄 서비스에서는, 현재의 Java 2D 인쇄 기능을 지원하면서, 다음의 점을 포함한 많은 기능이 향상하고 있습니다.

  • 클라이언트 및 서버 어플리케이션으로부터, 프린터 기능에 근거해 프린터의 검출과 선택을 실시해, 인쇄 작업의 프로퍼티을 지정할 수 있다. 즉, 이 JPS 에서는, 인쇄 하부조직에 부족한 컴퍼넌트인, 프로그램으로부터의 프린터 검출 기능을 실현하고 있다.
  • 표준 IPP 속성의 구현이, JPS API 내에 퍼스트 클래스 객체로서 짜넣어지고 있다.
  • 어플리케이션으로부터, JPS API 에 짜넣어지고 있는 속성을 확장할 수 있다.
  • 산 이외의 벤더가, 서비스 프로바이더 인터페이스를 사용해 독자적인 인쇄 서비스를 플러그 인 할 수 있다.
