JavaTMAccessibility

 문서의 목차
Java Accessibility API 에 의해, 개발자는, 몸의 부자유스러운 (분)편이 이용 가능한 Java 어플리케이션을 작성할 수 있습니다. Accessibility 를 이용 가능한 Java 어플리케이션은, 화면 리더, 음성인식 시스템, 및 점자 단말등의 사용자 보조 기능에 대응하고 있습니다.

상세 정보

  • Java Accessibility 유틸리티
    Java Accessibility 유틸리티는, 지원 기술에 의한 Java Accessibility API 의 이용을 지원하는 기능입니다.
  • Java Access Bridge for Microsoft Windows
    Java Access Bridge for Microsoft Windows 를 사용하면(자), Windows 네이티브인 지원 기술을 사용해, 지원 기술에 대응하는, Windows 상에서 실행되는, Java 어플리케이션에 액세스 할 수 있습니다.
