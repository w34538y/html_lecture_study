제 4 장

문서의 인쇄와 스트림 전달




어플리케이션으로부터 인쇄 작업을 송신하려면 , 인쇄 데이터를 처리할 수 있는 인쇄 서비스를 검색할 필요가 있습니다. 적절한 서비스를 검색하면(자), 그 서비스로부터 인쇄 작업을 취득해, 인쇄 작업을 서비스에 송신합니다.

이 장에서는, 이하의 조작에 의해 문서를 인쇄 및 스트림 전달하는 방법에 대해 설명합니다.

  • 전의 2 살의 장으로 설명한 속성과 드큐먼트후레이바를 사용해, 적절한 서비스를 검색한다
  • 인쇄 작업을 작성한다
  • 인쇄 작업을 프린터 또는 서비스에 송신한다
  • 인쇄 작업 또는 서비스에 이벤트를 등록한다
