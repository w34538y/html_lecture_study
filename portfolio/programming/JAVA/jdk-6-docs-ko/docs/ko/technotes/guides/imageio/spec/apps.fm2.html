3.2 고도의 기능

심플 단순한 어플리케이션의 경우에서는, 전의 섹션으로 설명한 메소드로 충분합니다. 그러나, Image I/O API 는, 선진적인 어플리케이션을 작성하기 위한 , 한층 더 고도의 기능을 제공하고 있습니다. 이 API 의 고도의 기능을 이용하기 위해서는, API 의 일부인 ImageReader 및 ImageWriter 클래스를 어플리케이션으로부터 직접 조작합니다.
