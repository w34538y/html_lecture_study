슈퍼 클래스가 외부화 가능한 경우의 외부화

개설

Externalizable 인터페이스를 사용하는 경우는, 외부화 가능 객체는, 객체 상태를 보존하기 위해서 writeExternal 메소드를, 객체 상태를 복원하기 위해서(때문에) readExternal 메소드를 구현할 필요가 있습니다. 게다가 외부화가 가능한 객체는, 그 상태의 보존과 복원을 위해서(때문에) 슈퍼타입과 협조해 동작할 필요도 있습니다.

이 예제는, 객체가 외부화 가능한 슈퍼타입 상태의 보존과 복원을 실시하는 방법을 나타냅니다. 외부화 가능한 객체의 슈퍼타입도 또 외부화 가능한 경우는, 서브 클래스의 객체는 스스로의 writeExternal 메소드와 readExternal 메소드중에서 슈퍼 클래스의 writeExternal 메소드와 readExternal 메소드를 호출합니다.
