직렬화와 전개

개설

어느 특정의 경우에, Java 의 객체가 직렬화를 사용해 객체를 보존할 때, 데이터를 읽어들이는 클래스의 버젼이, 데이터를 기입한 클래스의 버젼과 다른 일이 있습니다.

이 예제는, 직렬화 시에, 특정의 클래스에 의존하는 메소드를 사용하지 않고 , 호환성이 손상되지 않는 변경을 실시하는 방법을 나타냅니다.

전개를 취급하는 다른 예 모두와 같게, 전개 후의 클래스 (AClass)에는,serialVersionUID 필드를 정의할 필요가 있습니다. serialVersionUID 필드는, 오리지날 클래스 (AClass)에 serialver 커멘드를 실행해 취득할 수가 있습니다.
